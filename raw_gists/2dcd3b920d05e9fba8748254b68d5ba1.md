# 要約スニペット集
この gist は、要約プロンプトのスニペットを集めたものである。1-中見出し 1-パターンで書かれている。

利用者は、この gist の URL を与えた上でスニペットを指定する。あなたは指定されたスニペットを理解し、そのとおりの要約をせよ。

- スニペットが指定されない場合、利用者が欲しそうなパターンを提示して選ばせよ。選択肢には数字をつけ、数字を入力するだけで選べるようにせよ。また選択肢を選ばせる UI やツールは呼び出すな
- 明示的な指示がない限りは、平易に解説して
- 明示的な指示がない限りは、提案は一切するな

## A4 パターン
A4 1枚分で要約せよ。ここで A4 一枚とは空行を含めて 36 行以内のこと。

## 1-3-10 パターン
1行で要約し、3行で要約し、10行で要約せよ。各要約は毎回本文からつくることとし、要約から要約をつくる真似はするな。
