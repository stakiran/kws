# Cosense 最近のページ一覧をクリップボードにコピーする UserScript

## やりたいこと

Cosense（旧 Scrapbox）で、最近のページ一覧を以下の形式でクリップボードにコピーしたい。

```
[pagename1]
[pagename2]
...
```

## 使う API

`/api/pages/:projectname` でページ一覧を取得できる。

- 同じドメインから叩くので認証は不要(ログイン中ならプライベートプロジェクトでも通る)
- 主なクエリパラメータ
  - `limit`: 取得件数(最大 1000)
  - `skip`: スキップ件数
  - `sort`: 並び順。`updated` / `created` / `accessed` / `linked` / `views` / `title` など

## 基本形

プロジェクトの「設定」→「Page」の UserScript に貼り付けて使う。

```javascript
scrapbox.PageMenu.addMenu({
  title: 'Copy recent pages',
  image: 'https://i.gyazo.com/icon.png', // 好きなアイコン URL に
  onClick: async () => {
    const limit = Number(prompt('何ページ分コピーする?', '20'));
    if (!limit || isNaN(limit)) return;

    try {
      const project = scrapbox.Project.name;
      const res = await fetch(`/api/pages/${project}?limit=${limit}&sort=updated`);
      if (!res.ok) throw new Error(`API error: ${res.status}`);
      const data = await res.json();

      const text = data.pages.map(p => `[${p.title}]`).join('\n');
      await navigator.clipboard.writeText(text);

      alert(`${data.pages.length} 件コピーしました`);
    } catch (e) {
      alert('失敗: ' + e.message);
      console.error(e);
    }
  }
});
```

## カスタマイズ

### 更新日順 → 作成日順

URL の `sort=updated` を `sort=created` に変える。

```javascript
const res = await fetch(`/api/pages/${project}?limit=${limit}&sort=created`);
```

### ピン留めページを除外

`.map` の前に `.filter` を挟む。

```javascript
const text = data.pages
  .filter(p => !p.pin)
  .map(p => `[${p.title}]`)
  .join('\n');
```

`p.pin` はピン留め順を表す数値で、ピン留めされていないページでは `0` になる。`!p.pin` で「ピン留めされていないもの」だけ残せる。

## 両方適用した完成版

```javascript
scrapbox.PageMenu.addMenu({
  title: 'Copy recent pages',
  image: 'https://i.gyazo.com/icon.png',
  onClick: async () => {
    const limit = Number(prompt('何ページ分コピーする?', '20'));
    if (!limit || isNaN(limit)) return;

    try {
      const project = scrapbox.Project.name;
      const res = await fetch(`/api/pages/${project}?limit=${limit}&sort=created`);
      if (!res.ok) throw new Error(`API error: ${res.status}`);
      const data = await res.json();

      const text = data.pages
        .filter(p => !p.pin)
        .map(p => `[${p.title}]`)
        .join('\n');
      await navigator.clipboard.writeText(text);

      alert('コピーしました');
    } catch (e) {
      alert('失敗: ' + e.message);
      console.error(e);
    }
  }
});
```

## 注意点

- `filter` で除外するとコピーされる件数が `limit` より少なくなることがある。「ピン留め除外後にちょうど N 件欲しい」場合は、多めに取得してから絞る実装にする必要がある。
- メニューではなくキーボードショートカットで起動したい場合は `scrapbox.PageMenu.addMenu` ではなく `document.addEventListener('keydown', ...)` で実装する手もある。
