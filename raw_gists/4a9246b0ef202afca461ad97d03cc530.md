# サイレント退職と「静かな退職」

## 結論

両者は**別の概念**。日本語では似ているが、起きていることはほぼ正反対。

- **静かな退職 (Quiet Quitting)** → 辞めない。在籍したまま最低限の業務だけこなす
- **サイレント退職 (Silent Quitting)** → 辞める。前触れなく突然離職する

---

## 比較表

| 観点 | サイレント退職 | 静かな退職 (Quiet Quitting) |
|---|---|---|
| 実際に辞めるか | 辞める（物理的に離職） | 辞めない（在籍し続ける) |
| 現れ方 | 前触れなく突然退職届／退職代行から連絡 | 最低限の業務だけこなす働き方が続く |
| 本人の状態 | 不満やストレスを抱え込み、限界で爆発 | 仕事と心理的に距離を取ってバランスを保つ |
| 主な原因 | 組織側のサポート・コミュニケーション不足 | 個人の価値観、ワークライフバランス重視 |
| 企業へのダメージ | 大（人が消える、業務引継ぎ不能） | 中（エンゲージメント低下、周囲の負担増） |
| 気づきやすさ | 本人が告げるまでゼロ、退職時点で手遅れ | 兆候はあるが気づかれにくい |

## サイレント退職の出所

**明確な単一の原典がない**。日本のHR業界・メディアで2024年頃から使われ始めた、和製英語に近い用法。英語圏では "silent quitting" は "quiet quitting" の同義語として使われることもあり、用語として完全には独立していない。

### 日本での使われ方

- Smart相談室などのメンタルヘルス支援事業者が「事前の予兆や相談がないままに突然起こってしまう退職」と定義
- 人事領域で「退職代行から急に連絡が来る」「普段通り働いていた社員が唐突に退職届を出す」といった現象を指す

### 背景にある構造

1. **退職代行サービスの普及**
   - マイナビ2024年調査: 直近1年間に転職した人の16.6%が退職代行を利用
   - 20代の利用率は18.6%と最も高い
   - 利用理由トップ3:
     - 「引き留められた（引き留められそうだ）から」 40.7%
     - 「自分から退職を言い出せる環境でないから」 32.4%
     - 「退職を伝えた後トラブルになりそうだから」 23.7%

2. **終身雇用・年功序列の衰退**
   - 転職がネガティブでなくなった
   - ミレニアル世代・Z世代の「個性・キャリアアップ重視」の価値観

3. **「相談しても無駄」という諦め**
   - 職場環境への不満があっても「言ったところで変わらない」と判断
   - 黙って去る選択が合理的になっている

### 最近の動向（2026年）

- 東京商工リサーチ調査: 退職代行経験企業は大企業15.7%、中小企業6.5%
- 企業側も防衛策を強化:
  - 「利用歴が分かれば採用に慎重になる」 49.3%
  - 「利用歴が分かれば採用しない」 26.0%
  - 約4社に3社がマイナス評価

## 位置づけの整理

| | 静かな退職 | サイレント退職 |
|---|---|---|
| 概念の性格 | 国際的な学術・調査概念 | 日本の実務用語 |
| 背景にある事情 | ハッスルカルチャーへの反動、ワークライフバランス志向 | 退職代行、Z世代の離職観、終身雇用の崩壊 |
| 裏付け | Gallup、学術調査、各国メディア | 日本国内の人事コンサル・HR系メディアの観察 |

「静かな退職」はGallupなど国際的な大規模調査で裏打ちされた概念。一方「サイレント退職」は日本の労働市場の固有事情（特に退職代行の普及）を背景に生まれた実務用語、と位置づけると整理しやすい。
