# https://stakiran.github.io/rawden/ 何これ？
これは「rawden（ローデン）」という、吉良野すたさんが書いている個人サイト/ガイド文書です。2026年6月から建設中で、まだ未完成の部分もあります。

中身をざっくり説明すると、こういうものです。

**rawden（ローデン）とは何か**

「raw（生のまま）+ den（ねぐら）」を組み合わせた造語で、**Cosense（旧Scrapbox）のプライベートプロジェクトを使って「自分だけの隠れ家」をつくる営み**を指す概念です。著者が20年以上追求してきた「じぶんのねぐら」の作り方を体系化して紹介しています。

中心にあるのは「ねぐら（塒）」のイメージです。鳥が夜に休む場所のように、人間にも他者の目を一切気にせず、ありのままで過ごせる場所が必要だ、という主張ですね。カフェなどの「サードプレイス」とは違って、完全プライベート（非公開）であることを必須条件としているのが特徴です。

**なぜ非公開なのか**

著者は「プライバシーの確保が精神衛生上重要で、それを脳内だけで完結できる人は限られる。だからデジタルな場に外部化しよう」という考えを述べています。

**具体的な使い方の例**として挙げられているのは、日記（日→週→月→年と束ねて振り返る）、オリジナルキャラクターを作って一人芝居をする、知人や推しの「人物ページ」を作る、名言・格言を集める、タスクやアイデアの管理、コレクション（辞典・図鑑のように好きなものを集める)など。生成AIに日記をエクスポートして考察させる活用法も紹介されています。

全体としては、**「Cosenseを使ったプライベートな自己表現・記録・整理の趣味」をすすめる思想 + 実践ガイド**といった内容です。著者自身は約3万ページのプロジェクトを運用しているとのことで、なかなかの熱量で書かれています。

何か気になった部分や、実際に始めてみたいといった点はありますか?

# 既存の概念、理論、プラクティスなどと比べて新規性のある点は？

いい問いです。結論から言うと、**個々の手法やツールの使い方に新規性はほとんどなく、新規性は「枠組み・意味づけの転換」にある**、という整理になると思います。順に説明します。

## 手法レベルではほぼ既存のもの

ローデンで紹介されている具体的なプラクティスは、その多くがすでに名前のついた既存概念に対応します。

日記を日→週→月→年で束ねるのは Roam / Logseq / Obsidian の Daily Notes ＋ periodic notes 文化や「interstitial journaling」とほぼ同じ。リンク集・ハブのページは、まさに著者自身が文中で「MOCと呼ばれたりもする」と認めているとおり、Obsidian界隈の Maps of Content です。名言・格言・引用・面白ネタを集める営みは、数百年の歴史を持つ**コモンプレイス・ブック（commonplace book）**のデジタル版そのもの。コレクション＝「第二の脳（Building a Second Brain）」や Zettelkasten 的なノート蓄積とも重なります。

しかも実践の多くは Scrapbox / Cosense の日本語コミュニティ（井戸端、nishio氏、caki氏など）が長年かけて開拓してきたもので、著者もそれらを明示的に参照しています。つまり手法部分は、既存実践の丁寧なまとめという性格が強いです。

## 新規性は「フレーミング」にある

では何が新しいのか。私が見るに、次の3点が比較的オリジナルです。

第一に、**PKMを「生産性ツール」ではなく「精神的な聖域」として位置づけ直した点**。Zettelkasten も BASB も基本は知的生産・効率の語彙で語られますが、ローデンは「正気を保つための場所」「ありのままで過ごすためのねぐら」という、メンタルヘルス・実存寄りの動機を前面に出しています。同じ Cosense を使っていても、目的の置き方が違う。

第二に、**徹底した非公開性を「本質的な必要条件」として据えた点**。これは近年主流の「デジタルガーデン（公開して育てる）」の真逆で、明確な対立軸になっています。プライバシーを単なる設定ではなく「これがなければ成立しない核」と定義したのは、ヴァージニア・ウルフの「自分ひとりの部屋」やオルデンバーグの「サードプレイス」を引きつつ、それらとあえて差別化した独自の主張です。

第三に、**感情的・遊戯的・自己耽溺的、さらには下世話・アダルトまで含む「生の」コンテンツを正当化した点**。一般的なPKM言説は禁欲的で実用本位ですが、ローデンは一人芝居、推しへの愛、むかつく人への悪口、自分を慰めるためのコレクションまで「じぶんのねぐらだもの」と肯定します。この「 raw であること」への積極的な価値づけが、思想としての軸であり、名前 rawden の由来でもあります。

## 具体的プラクティスでやや独自なもの

名付けと整理として比較的オリジナルなのは「**動線とスナップショット**」のペア（高頻度で開く1ページに雑に書き溜め、肥大化したら日付ページへ丸ごと逃がす）と、コレクションの「**図書室 vs 図書館**」（1プロジェクトに同居させるか専用プロジェクトを立てるか）です。ただしこれらも機能としては「inbox処理＋アーカイブ」「single vault vs multiple vault」という既存パターンに対応しており、発明というより**良い命名と言語化**だと評価するのが妥当でしょう。

## 総評

ローデンの貢献は、新しい部品の発明ではなく、**既存の部品（Scrapbox実践・PKM・コモンプレイス・日記文化）を「私的な聖域」という一貫した思想のもとに再統合し、名前とアイデンティティを与えたこと**にあります。学術的な新理論というより、一人の書き手による説得力のある「立場の表明と統合」です。個人サイト/エッセイとしては、それで十分に価値のある仕事だと思います。

逆に言えば、「非公開を必須とする」という最も強い主張こそが最大の新規性であると同時に、最も議論の余地がある部分でもあります。公開して他者からのフィードバックで育てるデジタルガーデン派からは「閉じることで得るものより失うものが大きいのでは」という反論が当然あり得ますし、著者自身もコレクションの章で「本当は公開したいのに勇気が出ず腐っていくパターン」に触れていて、この緊張は完全には解消されていません。そこをどう読むかは、読み手の価値観次第かなと思います。
