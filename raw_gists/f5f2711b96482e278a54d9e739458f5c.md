# 認知アーキテクチャと環境耐性のフレームワーク

「犬や猫に邪魔されながら仕事できる人が理解できない」という問いから出発し、なぜ人によって他者の存在を許容しながら仕事できるか／できないかを、認知科学的に整理する。

---

## 1. 三軸モデル：Memory / Attention / Sensor

人が仕事中にどれだけ外部からの割り込み（ペット、子供、パートナー、同僚など）を許容できるかは、以下の三つの認知要素の組み合わせで決まる。

### Memory（ワーキングメモリ容量）

- 頭の中に一時的に保持できる情報の量と帯域
- 容量が大きい人は、タスクのコンテキストを保持したまま割り込みを別の領域で処理する余裕がある
- 容量が小さい人は、タスクでほぼ満杯になっており、割り込みが入ると保持中の情報を追い出さないと処理できない
- 学術的裏付け：ワーキングメモリ容量の個人差はマルチタスクの成功を予測する（Colom et al., 2010; Redick et al., 2016）

### Attention（注意制御）

- 不要な刺激を抑制し、必要な情報だけを通すゲート機能
- 注意制御が高い人は、犬の気配やパートナーの声を自動的にフィルタリングできる
- 注意制御が低い人は、入ってきた刺激を弾けず、すべてが処理対象になる
- 学術的裏付け：ワーキングメモリ容量がマルチタスクを予測するのは、注意制御（attention control）との共有分散によるところが大きい。「スーパータスカー」は前帯状皮質と前頭極皮質を効率的に動員し、目標維持と妨害抑制を行っている（Medeiros-Ward et al., 2015）

### Sensor（感覚処理感受性）

- 環境からの刺激をどれだけ深く処理するかの個人差
- 感受性が高い人（HSP）は、他者の存在自体が大きな情報量として入力される
- 感受性が低い人は、同じ環境でも刺激が「登録」されにくい
- 学術的裏付け：感覚処理感受性（SPS）は人口の15〜20%に見られる生得的特性。SPSが高い人は島皮質と前頭前皮質の活性化が増大しており、社会的刺激に対してより多くの認知リソースを消費する（Aron et al., 2010）。SPSの高さと内向性は、孤独を求める動機を有意に予測する（Thomas et al., 2024）

### 三軸の相互作用

三軸は独立ではなく、掛け算で効く。

- Sensor が高い → 入力過多 → Attention の負荷が上がる
- Attention がフィルタリングに消費される → Memory の実質的な可用領域が減る
- Memory の帯域が狭い → Attention をより厳密に運用する必要がある

結果として、Sensor が高く Memory が小さい人は、Attention を総動員しても追いつかない状態になりやすい。環境の徹底的なコントロールは「好み」ではなく「認知アーキテクチャ上の必然」となる。

## 2. CPPF：他者の存在コストの序列

CPPF（Child, Pet, Partner, Friend）は、生活に他者を受け入れるコストの序列を表す概念。三軸モデルとの対応は以下の通り。

| 対象 | Sensor負荷 | Attention負荷 | Memory負荷 | 特徴 |
|------|-----------|--------------|-----------|------|
| Child（子供） | 最大 | 最大 | 最大 | 予測不能、大きな音、感情的応答を要求 |
| Pet（ペット） | 中程度 | 中程度 | 低〜中 | ある程度予測可能だが身体的存在感がある |
| Partner（パートナー） | 中程度 | 低〜中 | 低〜中 | 学習可能でルール化できるが感覚的存在は残る |
| Friend（友人） | 低 | 低 | 低 | 時間を限定でき、最もコストが低い |

CPPFの順序は、三軸への負荷の大きさの順序とほぼ一致する。

## 3. IQ・ASD・ADHD・アファンタジアと三軸の関係

### IQ（知能指数）

- IQ はワーキングメモリ容量と中程度の相関がある
- 三軸の処理能力の「天井」を上げるが、Sensor には関与しない
- IQ が高いほど処理の深さが増すため、一つのタスクに投入するリソースが大きくなり、割り込みで失うものも大きくなる可能性がある
- **IQ は三軸の上限を決めるが、「環境耐性」を上げるとは限らない**

### ASD（自閉スペクトラム症）

三軸すべてに独特の形で関与する。

| 軸 | 影響 |
|----|------|
| Sensor | **高い。** 感覚過敏はASDの中核的特徴の一つ。音、光、人の気配がすべて過剰に入力される |
| Attention | **硬直的。** フィルタリングが苦手な一方、興味対象への過集中（ハイパーフォーカス）は極端に強い。配分の柔軟性が低い |
| Memory | **個人差が大きい。** 特定領域では非常に高い保持能力を示すことがある |

結果として、ASD の人は「自分が制御した環境では異常に高い集中を発揮するが、制御できない環境では急速に機能低下する」というプロファイルになりやすい。環境の完全なコントロールを必要とする。

### ADHD（注意欠如・多動症）

Attention に最も直接的に影響する。

| 軸 | 影響 |
|----|------|
| Sensor | **必ずしも高くない。** 入力量は普通のケースがある |
| Attention | **制御不全。** ゲート機能がうまく働かない。刺激が入ってきたときに「無視する」抑制が効きにくい |
| Memory | **容量自体は普通だが、** Attention の不安定さのせいで実質的に使える領域が減っている |

逆説的な傾向として、ADHD の人は「静かな環境でも集中が難しい」一方、「適度な刺激がある方がむしろ Attention が安定する」場合がある。カフェの方が集中できる、BGM がある方がいい、というのはADHD的なプロファイルに多い。

### アファンタジア

Memory への影響が最も大きい。

| 軸 | 影響 |
|----|------|
| Memory | **帯域が狭い。** ワーキングメモリの視空間スケッチパッドが利用制限されており、言語・抽象チャンネルに一極集中。使えるチャンネルが実質一つ少ない |
| Attention | **要求が上がる。** 使える帯域が狭いからこそ、入れるものの選別をより厳密にしなければならない |
| Sensor | **間接的に負荷が上がる可能性。** 内的な視覚イメージがない分、外部からの感覚入力が相対的に前景に出やすい |

アファンタジアの人は視覚的なイメージとして情報を「置いておく」ことができないため、言語チャンネル上で保持していた抽象的な構造が割り込みで崩れると、再構築のコストが高い。

## 4. プロファイルの複合効果

### 高環境耐性プロファイル（ペット・家族と暮らしながら仕事可能）

- Sensor：低（刺激がそもそも「登録」されにくい）
- Memory：大（容量に余裕があり並列処理が可能）
- Attention：柔軟（自動的にフィルタリングが機能）
- 割り込みは「コンテキストスイッチ」ではなく「並列処理」として処理される

### 低環境耐性プロファイル（一人でないと仕事にならない）

- Sensor：高（入力過多）
- Memory：小（タスクでほぼ満杯）
- Attention：硬直的または総動員状態（余剰リソースなし）
- 割り込みは致命的。環境の完全なコントロールが生存戦略となる

### 例：IQ96 + ASD + アファンタジア

- Memory：天井が平均域（IQ96）、かつ視空間チャンネルが使えない（アファンタジア）→ 実質帯域が非常に狭い
- Sensor：高い（ASD の感覚過敏）→ 入力過多
- Attention：硬直的（ASD）→ 過集中は強いが柔軟な配分が苦手

三軸すべてが「環境に余裕がない」方向に振れており、一人で静かで完全にコントロールされた環境を必要とするのは認知アーキテクチャ上の必然。

## 5. 仕事のスタイルへの拡張

この三軸モデルは、ペットや家族との共存だけでなく、仕事のやり方の好みにも拡張できる。

| 傾向 | 三軸プロファイル |
|------|----------------|
| モブプロ・ペアプロを好む | Sensor低、Memory大、Attention柔軟。他者の存在コストが低く、同期的やりとりが苦にならない |
| 非同期レビュー・一人作業を好む | Sensor高、Memory小〜中、Attention硬直的。テキストと一人で向き合う方が認知負荷が低い |
| カフェで仕事する方が捗る | ADHD的プロファイル。適度な外部刺激がAttentionの安定に寄与する |
| 完全な静寂でないと無理 | ASD的プロファイル。環境変数を極限まで減らすことで認知リソースをタスクに全振りする |

仕事のやり方の好みは、認知スタイルや特性と地続きであり、その人の人生全体の価値観（CPPFの受容度、コントロール欲求の強さ、同期vs非同期の選好）と同じ根から生えている。
