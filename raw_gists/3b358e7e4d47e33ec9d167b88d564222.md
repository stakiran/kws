めちゃくちゃポエミーな書き殴りだけど https://gist.github.com/stakiran/10ee1b269289de9733791b262be2c1f8
 に書いた。要するに、
 
- github.com にこもりたい
- Markdown で書きたい
- ブログっぽい見た目にしたい（Gist はコメント欄もあるようにブログっぽい）

## 投稿フロー

### 1. 下書き
下書きファイル stakiran.md に以下フォーマットで下書きを書く。

```markdown
# YYMMDD_HHMMSS わかりやすく検索しやすいタイトル
YYMMDD_HHMMSS_simple_and_understandable_filname.md

Body
...
```

### 2. 投稿する
github.com にログインして、[Create a new Gist](https://gist.github.com/) から 1 の内容を投稿。

### 3. リンクを張る
[stakiran's gist](https://gist.github.com/stakiran/8d01f34304c54851b7f136d181d04af1) に、2 へのリンクを張る。

## 今後
投稿フローを自動化したいなぁ。

- 1 の内容をクリップボードに入れといてから、以下を行うスクリを起動
  - クリップボードから投稿内容を取得＆パース
  - 2 は GitHub API 使えばたぶんいける
  - 3 は clone したディレクトリに対して操作すればいける
