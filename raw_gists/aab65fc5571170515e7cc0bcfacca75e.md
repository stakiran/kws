# LGBTとSOGIの違い

## SOGIとは

SOGI（ソジ）は **Sexual Orientation（性的指向）** と **Gender Identity（性自認）** の頭文字をとった言葉で、**すべての人が持つ属性**としての性的指向と性自認を指す枠組みである。

## LGBTとは

LGBTは以下の4つの頭文字をとった言葉で、**特定のセクシュアルマイノリティの人々**を指すカテゴリーである。

- **L** — Lesbian（レズビアン）
- **G** — Gay（ゲイ）
- **B** — Bisexual（バイセクシュアル）
- **T** — Transgender（トランスジェンダー）

LGBTQ+、LGBTQIAなど、さらに多くのアイデンティティを含む形で拡張されることもある。

## 両者の関係

SOGIは「分類の軸そのもの」、LGBTは「その軸上の特定のグループ」にあたる。

たとえるなら、**「血液型」がSOGI**に、**「A型・B型・O型・AB型」がLGBT**に相当する。血液型という枠組みは全員に関わり、その中のどれかに誰もが当てはまる。同様に、SOGIという「性的指向×性自認」の分類軸はすべての人が持っており、LGBTはその中でマイノリティにあたる組み合わせをまとめた呼び名である。

つまり、**SOGIが上位の枠組み、LGBTがその中の下位カテゴリー**という関係になる。

## LGBT以外のSOGIの例

### 最多数派

- **異性愛（ヘテロセクシュアル）かつシスジェンダー** — 生まれた時の性別と性自認が一致し、異性に惹かれる人々。人口の大多数を占める。

### LGBTの4つに収まらない性的指向

| 名称 | 概要 |
|------|------|
| アセクシュアル（Aセクシュアル） | 誰にも性的な惹かれを感じない |
| パンセクシュアル | 性別を問わず惹かれる |
| デミセクシュアル | 親しい関係を築いた相手にのみ惹かれる |

### LGBTの4つに収まらない性自認

| 名称 | 概要 |
|------|------|
| ノンバイナリー | 男女どちらにも当てはまらない |
| ジェンダーフルイド | 性自認が流動的に変化する |
| Xジェンダー | 男女の枠組みに当てはまらない（日本発の概念） |
| クエスチョニング | 性自認を特に定めない・模索中 |

## SOGIが使われる背景

LGBTという枠組みだけでは表しきれない多様なあり方が存在するため、LGBTQ+と「+」をつけたり、SOGIという包括的な枠組みが使われるようになった。SOGIは「マイノリティだけの問題」ではなく「誰もが当事者である」という視点を提供する。

近年では職場や教育の場で「SOGIハラ（SOGIハラスメント）」のように、性的指向や性自認に関するハラスメントを誰にとっても起こりうる問題として捉える際にも用いられている。
