# AIレイオフの罠 — 論文整理

**Hemenway Falk & Tsoukalas (2026)**

## 定義した問題

AIが労働者を経済が再吸収できる速度より速く置き換えると、企業が依存する消費需要そのものを蝕む。崖が全員に見えているなら、合理的で先見的な企業はなぜ崖に向かって突進するのか。具体的には3点を問う:(1) 合理性と完全な先見性だけで競争的な過剰自動化を防げるか、(2) 防げない場合に歪みの大きさを決めるものは何か、(3) どの政策手段がそれを是正するか。

## 定義した前提(ベースラインモデル)

労働市場ではなく**製品市場**に焦点を当てたタスクベースモデル(Acemoglu–Restrepo型を改変)。

- N≥2の対称的な企業が1セクターに存在。各企業はL個のタスクを持ち、自動化率 αᵢ∈[0,1] を選ぶ
- 自動化タスクはコストc、人間タスクは賃金w(c≤w)。1タスク当たり節約 s=w−c
- 統合摩擦は凸的:k/2·Lαᵢ²(後続タスクほど自動化が困難)
- 賃金は外生(ベースライン)。出力は各タスク1単位で固定(生産性・品質マージンを遮断)
- **需要側**:労働者は所得の割合λをセクター財に支出。所有者の支出はゼロ(MPC非対称性)
- 解雇された賃金所得の割合ηが再雇用・移転で回復、残り(1−η)wが失われる
- 1タスク当たり実効需要損失 ℓ=λ(1−η)w
- 一回限りの同時手番ゲーム、解概念はナッシュ均衡。全企業に完全な透明性を仮定

## 「実験方法」=分析手法とその結果

数値実験ではなく、**ゲーム理論的均衡分析と比較静学**(全証明は付録)。

**中心的結果(命題1・系1):** 競争が需要外部性の罠を生む。自動化する企業はコスト節約を全額得るが、生み出す需要破壊のうち自分が負うのは ℓ/N のみ(競争的価格付けで残りはライバルに転嫁)。よって各企業の利潤最大化自動化率は**強支配戦略**であり、協調最適水準を上回る。先見性だけでは崖を防げない。

- **過剰自動化のくさび** = ℓ(1−1/N)/k:Nとℓで増加、kで減少。独占(N=1)は外部性を完全に内部化、市場が断片化するほど歪みは最大
- 摩擦ゼロ(k→0)では**囚人のジレンマ**となり全員が全労働力を自動化
- **死荷重損失(命題2):** 労働者から所有者への移転ではなく、両者がともに損をする。協調最適にパレート支配される

**政策手段の評価(6種、効率ベンチマークで判定):**

| 手段 | 外部性を是正するか |
|---|---|
| 再訓練・アップスキル(η) | 部分的 |
| ベーシックインカム(A) | 不可(利潤水準のみ変更) |
| 資本所得税(t) | 不可(限界マージンに作用せず) |
| 労働者持分参加(ε) | 部分的(λ<1では完全には閉じない) |
| コース的交渉(M<N) | 不可(自己拘束的でない) |
| **ピグー的自動化税(τ)** | **是正できる(唯一)** |

最適税率 τ*=ℓ(1−1/N)。税収を再訓練に充てればηが上がりℓが縮小、税が自己限定的になりうる。

**頑健性の検証(拡張、いずれも外部性は消えず):** AI生産性向上(ϕ>1)は「赤の女王効果」でくさびを**拡大**。内生的賃金調整は外部性が作動する閾値を上げるがくさびは閉じない。自由参入、資本所得の再循環、不完全競争でも消えない。

## 含意

問題は「企業が労働者を犠牲にして利益を得る」ことではなく、**過剰自動化が両者を害する浪費**であること。事後対応(再訓練・所得支援)ではなく、自動化を駆動する競争インセンティブそのものに政策が対処すべき、と結論。
