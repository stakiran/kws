# q
個人ウェブサイトを自分の城として太らせて人生体験を上げたい。直近では個人ウェブサイト運営者が集まるコミケのようなイベントもあり、そこで披露もしたいと思っている（3ヶ月後）。

今日 /now という考え方を知った。/about にて「このサイトについて」を取り上げるのは定番だが、それと同じノリで、/now で「今やっていること」を取り上げるのである。SNS その他日記の内容ではなく、自分の今のフォーカスを紹介するページである。

さて、ここからが本題だが、このように特定の意図を持ったページは色々考えられると思う。/xxxx の形になるよね。これを single function static page（sfsp）と呼びたい。static とつけているのは静的なページに限定したいからだ。JavaScript やバックエンドなどでリッチコンテンツをつくることは許容しない。あくまでも HTML で表現できる程度のものである。

sfsp として他に何があるかを 20 個提案して。

# a
1. **/uses**  
   使っているハード・ソフト・サービス・作業環境（エディタ、フォント、PC、机など）。

2. **/setup**  
   作業机・部屋・持ち物の構成。/uses よりも「配置」や「運用」に寄せる。

3. **/projects**  
   進行中・休止中・完了した個人プロジェクトの一覧（短い説明＋リンク）。

4. **/ideas**  
   温めているアイデア置き場。実装前提じゃなくてもよく、方向性だけをメモ。

5. **/principles**  
   行動指針・価値観・判断基準（何を大事にして何を捨てるか）。

6. **/defaults**  
   自分の「デフォルト設定」集（連絡手段、返信ペース、予定の立て方、普段の優先順位）。

7. **/contact**  
   連絡先と連絡方法のガイド。問い合わせの種類別の推奨ルートも書ける。

8. **/colophon**  
   このサイトの技術構成・方針（ビルド方法、フォント、ホスティング、アクセシビリティ方針）。

9. **/changelog**  
   サイト更新履歴。「何をどう変えたか」を短く積み上げる。

10. **/status**  
   今の稼働状況（例：忙しい/余裕あり/返信遅め）。/now よりも対外的。

11. **/questions**  
   よく聞かれる質問への回答（FAQ）。自分向けの定型文集としても機能。

12. **/start**  
   初見の人向けに「ここから読む」導線。代表記事・自己紹介・おすすめ順。

13. **/links**  
   外部リンク集（影響を受けたサイト、友人サイト、ツール、コミュニティ）。

14. **/people**  
   自分が紹介したい人・サイト運営者の一覧（短い推薦文つき）。

15. **/reading**  
   読書中/最近読んだ本と一言メモ。SNS的感想ではなく「何を吸収しているか」。

16. **/listening**  
   最近聴いている音楽・ポッドキャスト。気分や作業との相性など短評。

17. **/watching**  
   観ている作品（映画・アニメ・YouTube等）と、今の関心軸。

18. **/learning**  
   学んでいること（言語、数学、デザイン等）と、今の教材・到達目標。

19. **/goals**  
   目標（今年/今季/今月）と進捗の置き方。数を絞ると /now と相性がいい。

20. **/wishlist**  
   欲しいもの・行きたい場所・やりたいこと。買うためというより「興味のレーダー」。

必要なら、イベントで「sfsp の見本市」っぽく見せるためのカテゴリ分け（自己紹介系/運用系/制作系/インプット系/対外コミュニケーション系）や、URL設計ルール（英語統一、複数形の扱い、末尾スラッシュ等）も一緒に整える案を出せます。
