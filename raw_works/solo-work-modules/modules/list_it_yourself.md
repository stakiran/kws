# LIY(List It Yourself)
**LIY(List It Yourself)** とは、リストを自らつくることを意味する言葉である。DIY のリスト版と考えるとわかりやすい。

## なぜ LIY なのか
リストには文脈があるからだ。

文脈無きリストはただの列挙でしかない。文脈とは、ただの列挙に意味と価値を見出すための鍵なのである。残念なことに、文脈を手に入れるのは容易ではない。物語やエッセイとは違って、通常、リストには文脈が詳しく記述されないからである。

文脈を手に入れるのは難しい:

- 自己啓発書を思い浮かべよ。良く出来た本なら章ごとに、リストによるまとめを記載しているはずだ。だからといって、本文を読まずにまとめだけ読めばいいかというと、そうではない。逆に本文を読み、理解しておけば、あとはまとめだけで十分なことも多い。要するに、書籍のレベルで提供されない限り、そのリストの文脈を知ることはできないのだ。
- 買い物リストを思い浮かべよ。書いた本人なら理解できるが、他人には理解できないだろう。ただの羅列にしか見えないか、せいぜい TODO リストとして漏れなく買うくらいの解釈しかできないはずだ。運が良ければ「カレーをつくりたいのだな」といった推測も立つかもしれないが、その程度である。ここで一つ考えてみてほしい。あなたの買い物リストを他人でも理解できるようにするために、あなたが一体どれだけの解説を行わねばならないかを。

リストの作成者が、同時にその文脈を添付するのは非現実的なのである。だからこそリストは他人にとっては使い物にならないことが多い。知的生産者であれば、自分の文脈を勝手に混ぜ込んで解釈できるが、それでも元のリストの本来の文脈（本来の意味や価値）には届かない。たかが知れている。

そういうわけで、文脈の伴ったリストを手に入れるなら――意味と価値のあるリストを扱いたいのなら、自分でつくるしかないのである。

## リストのつくりかた
ソロワークモジュール「LIY」そのものは、リストのつくりかたを扱わない。他のモジュールを参照せよ。たとえば List Kit が使えるだろう。
