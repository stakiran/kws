# ライフパラダイム
# 生活の枠組み。
# Life Paradigm.

# 例
# ライフパラダイムがシフトした例
- 遊動民 → 定住民
- オフライン → オンライン
- 毎日出社 → リモート

# 性質
# パラダイムが進むほど、
- Standard
    - 利便性が上がり、生活水準も上がる（上がれる人が増える）
    - 富裕層やカースト上位など力を持つ者以外も享受できるようになります<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- Solo
    - ひとりでも生きやすくなる（生きれる人が増える）
- Spectrum
    - 多様性が増す。あり方の解像度が上がる
    - たとえば働き方で見ると、リモートの台頭により「毎日出社」一色から「毎日出社」「ハイブリッド」「フルリモート」の三色になっています<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# ライフパラダイムが言っていること
- 3S が向上しない限り、ライフパラダイムがシフトしたとは言えない
    - 3S の向上無きシフトは見せかけにすぎません<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- パラダイムシフトさせたいなら、3S が向上するようなあり方や仕組みを考えればいい
