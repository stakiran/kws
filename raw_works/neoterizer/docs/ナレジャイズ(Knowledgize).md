# ナレジャイズ(Knowledgize)
# 知識、経験、文化、プロセスなど「暗黙知」や「ドメイン知識」を第三者に伝えられるよう言語化・漂白化すること。
# ナレッジ化。

# 背景
- 情報共有の重要性は言うまでもない
    - 特に最近ではいかに生成AIに食わせられるか次第ですので、食わせられるように言語化できる必要がある
- しかし現代であっても全然できていない
    - 根本的な原因の一つに「そういう発想はない」があります。[エンジニアリング・マイノリティ](エンジニアリング・マイノリティ.md)と同様、言語化や共有の営みもまたマイノリティなのです。オープンに親しんでいる一部のITエンジニアやインターネットユーザーなどはその価値がわかりますが、マジョリティはそうではありません。かわりに、[📝グルーミング](📝グルーミング.md)など対面的かつ非言語的な営みを重視して非言語情報の洪水を浴びることでカバーしようとしますが、これも周知のとおり適性がありますし、[公然の職権乱用](公然の職権乱用.md)が起きやすいですし、「ノリ」が形成されるので政治が生まれやすい等問題は多数あります。だからこそ、もっとカジュアルにナレッジを扱えるようにならねばいけないと考えています<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- 一応[SECIモデル]のような「暗黙知の形式知化」の理論は多数あるが、理論的かつスケールも大きくなりがちで、カジュアルに取り入れられるものではない
    - 私がここがもったいないと感じています。ナレッジ化というと、とても重たい営みに聞こえてしまいますし、「ナレッジ」自体にも高い水準を求めがちです。そんなことはないのです。たとえば[🙏仕事術](🙏仕事術.md)もれっきとしたナレッジです。いわば「ナレッジ崇高神話」が蔓延しており、食い止めたいのです<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- カジュアルに万人に取り入れてもらうための概念が必要だろう、ということでナレッジの言葉を使って構成しました<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 概要
- 第三者に伝えるとは:
    - [知的生産](知的生産.md)の定義でもあるが、自分以外の第三者でも理解できる程度に整えること
    - 言語化と漂白化がある
        - 言語化：言葉で表現すること。またわかりやすさや厳密性などのバランスを考えた整理も行うこと
        - 漂白化：第三者には渡せないセンシティブな情報を取り除くこと。≒抽象的に表現すること
- 何をナレッジ化するか
    - 暗黙知。知として存在しているが、現状言語化されていないもの
    - ドメイン知識。社内や顧客の事情など、特定のドメイン（領域）に閉じた固有の暗黙知または形式知のこと。機密文書などすでに形式知として整っていることもある

# 意義
- ナレジャイズの概念が生まれたことで、いつ誰がどのようにナレジャイズするか、という議論が可能になったこと
    - たとえばナレジャイザーという役割を定義して、週に一度、チームやプロジェクトの情報を全社的に公開することができます。あるいはナレジャイズ・タイムなる時間帯を定義して、毎週金曜日の午前どこか1時間をナレジャイズにあてさせるといったことも思いつきますね。これらは単にナレジャイズの役割や頻度を議論しているだけです。ナレジャイズという概念がないと、このような議論自体が行えません＝発想を持つことすらできません<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
        - あるいはSECIモデルを取り入れよう……のような大げさな取り組みになってしまうでしょう

# Q&A
- Q: [知的生産](知的生産.md)との違いは？
    - Ans: 結構ややこしそうです
    - 知的生産はNeoterize（ネオタライズ）であり、一言で言えば造語することです。新しい概念をつくりだすことにフォーカスしています。一方、ナレジャイズは単に既存の暗黙知または形式知を、第三者に見せれるレベルにまで整えることを意味します
    - しかし、知的生産には「既存のものに名前をつけるだけ」の軽いものもあり、この点ではナレジャイズと被っています。つまり知的生産はナレジャイズにも役に立ちます
- Q: ナレジャイズという概念自体はナレジャイズですか？
    - Ans: 「いいえ」だと思います
    - ナレジャイズという概念自体は知的生産だと思います。背景でも述べたとおり、マジョリティにとっては新しい考え方だろうからです
