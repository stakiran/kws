# DCPC
# [SI]で扱うもので、ドキュメント、コード、パラメーター、コンフィグの総称。

# 対応
- ドコード
    - Document ドキュメント
    - Code コード、スクリプト
    - ドキュメントのとおりにコードを実装する
- パラコン
    - Parameter 設計上の値
    - Configration 実機上の値
    - パラメーターのとおりにコンフィグを設定（構築）する
