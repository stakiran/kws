# Clear Collaboration
# 非同期かつ[トピック指向](トピック指向.md)で情報共有と議論を進めていくという[探索駆動](探索駆動.md)手法。
# 略称はcc。

# 概要
- 非同期かつトピック指向で情報共有や議論を進めていく手法
    - ＝書く
- クリアを目指す
    - クリアとは以下の二点:
        - クリアにする（明らかにする）
        - クリアする（解決する）
    - 書くことで明らかになり、明らかになれば解決へと向かっていく
- ccでは、実際に行動する「ために必要な情報や意思決定を揃えておく」ところまでやる
    - [* ブースター]をつくると表現する
- トピックは Talk（ただの雑談） → Target（当事者を決める） → Task（期限やフォローを決める）と進めていく
- 役割分担をする
    - 当事者と助言者に分かれる
    - 音声で残す者、催促する者、発散する者や収束する者など各自の得意で支え合う
