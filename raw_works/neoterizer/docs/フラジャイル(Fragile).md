# フラジャイル(Fragile)
# タフの対義語で、脆いこと。
# 特に人にかかる言葉。

# 用例
- フラジャイルな人

# 概要
- 以下のタイプがある
    - Lower。キャパシティが低い。運動や栄養等の不足、障害や病気など。
    - Sensitive。キャパシティは悪くないが、燃費が悪いか感度が敏感。先天も後天もある。[HSP]や[ASD]など。
- フラジャイルであるとは脆弱である、とも言える
    - システムの脆弱性がそうであるように、弱点として常に存在している
    - つくりとしてそもそも弱い（Lower）、つくりはそうでもないが皮がえぐれて神経が表出している（Sensitive）、また両方もありえる
    - 基本的に自然回復はありえないし、治療が通用するとも限らないし、通用しても完治するとは限らない

# レベル
- タフ。[トリップロード(Trip Load)](トリップロード(Trip_Load).md)に連続で耐えられる
- ノーマル。連続では耐えられないか、[パフォーマンス]が明らかに落ちる
- フラジャイル。単一回でも耐えられないか、パフォーマンスが明らかに落ちる
