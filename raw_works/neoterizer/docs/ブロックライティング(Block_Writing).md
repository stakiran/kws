# ブロックライティング(Block Writing)
# 知的生産のやり方の一つで、箇条書きの塊（ブロック）をつくる・並べるもの。

# 例
<img src="https://gyazo.com/f49ce0bf6745b3d879ab0f41337d53c7.png" />

- 赤色がブロック
    - s(サマリーブロック)、k(検討ブロック)などが並ぶ
    - 区切り線もブロック
- オレンジはサイクル
    - 一つのサマリーを伴った、きりのいい単位
    - 見づらいので区切り線で区切り
    - 例では下がサイクル1で、上がサイクル2。サイクル2はサイクル1のサマリーを踏まえている
