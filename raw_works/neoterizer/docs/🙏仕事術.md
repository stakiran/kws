# 開発秘話
- これは要するに[知的生産物]なるもの、特にビジネスや人生に役立つものをどう呼ぶかという問題なのです。既存の上手い言葉はありません。ライフハックも自己啓発も仕事術もごく限定的ですし、何より自らつくるとの発想がありません。最初は造語していたのですが、当然わかりづらいです。ワークウェア、ワークハック、実用文学、実用的空想、実用創作、ノートプロトタイピング、バーバルプロトタイピング――色々つくりましたが、伝わるわけがない。現在でもまだ解はないですが、ひとまず仕事術に目をつけて、上書きすることにしました。実際、仕事術というと「手続き」を思い浮かべると思いますが、その手続きをこなすためにはいろいろな前提をクリアせねばなりません。結局、考え方やあり方といった目線も必要になります。であれば、仕事術という言葉に、これらの意味も混ぜてしまっていいだろう、ということでこうなっています。
