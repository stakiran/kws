# 🔗イズミング(ISMing)
# イズムをつくること。

- クリティカル。批判や離脱を目的としたもの
    - [画一主義(Uniformism)](画一主義(Uniformism).md)
- プロポーサル。次に行くための望ましいあり方 or 先端を行く人達のあり方の言語化
    - [ページズム(Pagism)](ページズム(Pagism).md)
    - [ベーリズム(Veilism)](ベーリズム(Veilism).md)
    - [デリケーティズム(Delicatism)](デリケーティズム(Delicatism).md)
- アタック。知的生産者として読み手を震わせる革新的・変人的なあり方
    - [ダシズム(Dashism)](ダシズム(Dashism).md)
    - [ディーピズム(Deepism)](ディーピズム(Deepism).md)
    - [インパクティズム(Impactism)](インパクティズム(Impactism).md)
- 未分類
    - [リズミズム(Rhythmism)](リズミズム(Rhythmism).md)
