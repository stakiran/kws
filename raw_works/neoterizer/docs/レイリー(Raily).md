# レイリー(Raily)
# レールに沿っているさま。
# 特に予定やスケジュール、プロセス、上位者、顧客など絶対的に従うべき対象がいてその奴隷になっているさま。

# 別の表現:
- [プロセス原理主義]
    - [📖新 企業の研究者をめざす皆さんへ]の言葉
    - 厳密なプロセスやルールを導入することで、現場のエンジニアが自分で考えなくなってしまう
    - そのような会社では、独創的な仕事ができなくなってしまう
    - プロセスに基づく品質管理の考え方
        - 「あるプロセスに基づいて作られたソフトウェアは一定の品質を満たすと推定する」
    - 今はこういうことが起こってる:
        - レビューに通らないと先に進めないので、「レビューに通りやすい成果物」をつくるようになる
        - レビュワーも人間なので、分厚い観点やルールにしたがって調べきるのは無理
            - ここを重点的にチェックしよう、など傾向が出る
