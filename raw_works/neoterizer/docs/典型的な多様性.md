# 典型的な多様性
# [多様性]として想定される「典型的なもの」。
# 無論、多様性は典型的なもの以外もあるが、典型的なものだけが扱われがちである。
# そうではないのだと強調するために、今扱われているものに「典型的な多様性」の名前をつけた<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 例
- 女性
- 育児、介護
- 性的マイノリティ
- 身体障害
