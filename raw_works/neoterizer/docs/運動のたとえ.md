# 運動のたとえ
# [公然の職権乱用](公然の職権乱用.md)、特にコミュニケーションに関するものをわかりやすくたとえたもので、人間に必要だからといって仕事に必ず取り入れねばならない道理はないということ。

# 概要
- 運動は人間にとって重要である
    - 運動は根源的な欲求の一つでもある
    - ある程度は運動しないと身体が壊れる
- 昔は運動も公然の職権乱用がされていた
    - 精神論や根性論に相当するものである
    - 例: 「新人はエレベーターを使わず階段を登れ！」
- しかし現在では運動は以下の塩梅になっているだろう
    - 1. あえて行う必要はないし、できるだけ省くべきだ
        - 乗り物やリモートなど
    - 2. しかし人間としては必要だが、それはプライベートで各自が解決せよ
    - 3. ただし会社として運動系のイベントを催すくらいは良い
    - これを[DIIP(Do It In Private)原則](DIIP(Do_It_In_Private)原則.md)と呼びます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- 本題に入るが、ではコミュニケーションはどうだろう？
    - 運動と同じである
    - コミュニケーションもDIIPすればいいのである

# つまり「コミュニケーションもDIIPせよ」をわかりやすく表現したのが、運動のたとえである。
