# 違い
- Q: [Activity Based Working]の違いは？
    - Ans: 全然違います
    - ABW
        - 状況（特に自分の気分など主観的な状況）次第で働く場所を変えるという考え方です。特にオフィス内に閉じたもので、最近、でもないですが、フリーアドレスメインにしつつもコミュニケーションエリアや集中ブースやその他色んなブースや席があるような、あのオフィスレイアウトを指します
        - 無論、ABWの名前から考えると、オフィスに閉じる必要はないのですが、実際には閉じられます。[「仕事内容に応じた働き方」の実践基本ガイド：知っておくべきこと https://www.wework.com/ja-JP/ideas/research-insights/expert-insights/essential-guide-activity-based-working]などがわかりやすいですが、「アクティビティ」の定義を会社に委ねると（会社としては物理的に集めて搾取した方が生産性が上がりますから）そうなりがちです
    - ABNにおけるアクティビティは、単に「何をした」を意味します。この「何をした」をベースとしてつながりをつくろうというものです。整備中なので中身はまだこれからです
