# 認知的生産
# 自ら意思決定を行うこと。

# 名前
- [知的生産](知的生産.md)をもじった
- 認知的に何かをつくっているに等しい、と言える程度の負担はあると思いました<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
    - 実際知的生産含むものづくりが誰にでもできるものではないように、認知的生産も誰にでもできることではありません
    - 意思決定というとマネジメントや経営層の日常と思われがちですが、[リアクティブワーク(Reactive Work)](リアクティブワーク(Reactive_Work).md)として処理する人も多いです
        - 悪いことではない。決定やリードは必要
    - ただ、そのような受動的な決定ではまかなえない領域があるということです
        - 自分なりに考え抜いて、それをわかるように言語化して、伝える――
        - 自分が傷つく覚悟をもって（腹をくくって）意思決定する営みが必要です

# 背景
- このような営みは、わかるようで意外とわかりません<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
    - 啓蒙が必要です
    - そこで知的生産にからめて、認知の言葉で名付けてみました
