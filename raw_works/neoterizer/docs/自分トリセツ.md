# 自分トリセツ
# 自分の働き方について記した文章や文書。

# 概要
- 働き方とは拘束の仕方を指す
    - 勤務時間、出社の頻度、会議の頻度など

# 使い所
- [道具としてのメンバー(Member As A Tool)](道具としてのメンバー(Member_As_A_Tool).md)など、ある種割り切った使い方をしたい場合
    - どこまで使っていいかの合意を取るためにトリセツをつくる

# 例
# （ここで未満とは「これよりも低頻度」を指す。週1未満 → 週1よりも低頻度）
- 出社
    - Must 定期的な出社は行わない
    - Must 必要に応じて行う
    - Should 出社頻度は月1未満
        - ただし一時的に週1レベル（月4～5回）になるのは問題無い
- 勤務
    - Must スーパーフレックスを使う
        - いつ勤務するかはその日次第
        - 会社要件に基づき、月140時間以上勤務する
    - Should コアタイムは 11:00～13:00 とする
        - これ以外の時間帯はフレックス次第で勤務しないことがある
        - ただし調整してもらえれば融通は利かせる
    - Should 月残業は5H以内
- 会議
    - Must プロジェクトあたり週1未満
    - Should 1日あたり2回まで
    - Should セクション（午前・午後の単位）あたり1回まで
- イベント
    - Should 当日準備撤収時間を含め 2h 以上のイベントについては、移動時間含めて1日をフルで使う
