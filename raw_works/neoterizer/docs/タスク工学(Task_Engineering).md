# タスク工学(Task Engineering)
# 「タスク」を扱う工学。

# 背景
- [🙏タスク管理](🙏タスク管理.md)として知られるジャンルが昔からあるが、2025年現代でもまともに整備されていない
- そのせいで人間の自律性を拡張できず、原始的な働き方がいまだに蔓延している
    - たとえば出社回帰を唱えることしかできない
- タスク管理を整備・啓蒙できればいいが、この言葉にはすでに強いニュアンスがあり、整備はともかく、啓蒙は用意ではない
