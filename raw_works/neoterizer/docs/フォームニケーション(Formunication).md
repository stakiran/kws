# フォームニケーション(Formunication)
# フォームを用いてコミュニケーションすること。
# Form + Communication からきている。

# 概要
- 名前にはコミュニケーションが含まれているが、わかりやすさのための仕掛けにすぎない
    - 実際は[コンソリデーション(Consolidation)](コンソリデーション(Consolidation).md)である
- やり方:
    - フォームを一つつくり、そこに関係者が何でも情報を投稿する
    - 同時に投稿結果の共有も開放し、関係者全員がいつでも見れるようにする
- つまり、全員が投稿結果を見つつ、思いついたことを何でも投稿していく

# Q&A
- Q: 匿名？実名？
    - Ans: 匿名が良い
    - 情報を何でも集めたいので、実名はノイズでしかない
- Q: [クローズドな匿名](クローズドな匿名.md)が良い？
    - Ans: はい
    - 言い換えると、不特定多数向けの情報収集としては向いていない
        - もちろんただの匿名フォームは可能だが、フォームニケーションはコンソリデーションであり、徹底的な情報の集約を目指す。不特定多数ではその域には至れない
