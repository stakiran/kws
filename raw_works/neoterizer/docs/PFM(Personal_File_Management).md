# PFM(Personal File Management)
# 個人ファイル管理。
# 個人が持つファイルをどう管理するかのやり方、考え方、あり方。

# 概要
- 具体的には、ローカルPC上で扱うファイルをどのように[* 統一的に]管理するかの体系を指す

# 例:
- [DDWAD](DDWAD.md)
- [📝HBF+A](📝HBF+A.md)
- [📝PARAメソッド](📝PARAメソッド.md)
    - 一応挙げさせていただきましたが、個人的にPFMに含めることには懐疑的です。というのも、PFMとはローカルPCという「家」をどうつくるかという営みであり、これには「PCのセットアップ」という家自体の構築も含みます。よって、インストールアプリやその設定を管理するとの観点を重視しなければなりませんが、PARAメソッドは汎用的すぎてそこまで踏み込めていません<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
