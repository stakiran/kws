# 軽量タスク管理
# [🙏タスク管理](🙏タスク管理.md)はごく一部のみを扱うと割り切り、軽量な手法で管理すること。そのような手法。

# 背景
- [🙏タスク管理](🙏タスク管理.md)は広域かつ難解であり、理解や習得は難しい
- 一方、部分的に取り入れるだけでも役に立つ
- 現実を見ても「必要な部分だけコンパクトに使えるようにする」とひとまずやりやすい
    - そのような手法をつくればいい

# 概要
- 軽量の定義
    - 厳密な定義はないが、A4一枚やスライド一枚で概略を説明できる程度のボリュームを想定
