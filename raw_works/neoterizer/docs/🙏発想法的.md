# 🙏発想法的
# アイデア出しのような営みを指す。形容詞。

# 以下3要素から成る
- 発散。情報を出す
    - 何でも出す
    - [ダシズム(Dashism)](ダシズム(Dashism).md)も要求される
- 収束。情報を整理する
    - 分類、ラベリング、要約など整理の仕方は様々
- 蒸留。本質を導く
    - 発散収束内容はヒントにするだけであり、すべてに則らねばならないわけではない

# 背景
- 早い話、蒸留ができるのならば手段は問わない。しかし、蒸留は困難な道のりであり、発散と収束を行き来して情報を出し切り、思考の死角もなくしきらなければ到達できないし、それでも到達できるとは限らない。元来、蒸留は才能あるクリエイターの特権であったが、ここに再現性を持たせたのが発想法である――と私は理解している<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
