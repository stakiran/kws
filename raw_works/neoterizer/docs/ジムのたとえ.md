# ジムのたとえ
# [ASD]にとってコミュニケーションイベントは、「ジムに行く」レベルで負荷が高いということ。

# 詳細
- コミュニケーションイベントとは
    - 時間（と場所）を拘束する、集中的なコミュニケーションの機会
    - [飲み会]、打ち合わせ、その他コミュニケーションの多いイベント全般
