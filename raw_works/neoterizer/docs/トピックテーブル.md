# トピックテーブル
# [トピック指向](トピック指向.md)的なテーブル。
- ある一つの話題を扱うテーブルを指し、同座する者はその話題について話し合う

# 背景
- 一部の[ホールシステム・アプローチ]では「1-テーブル 1-話題」の運用を行う
    - 例: [オープン・スペース・テクノロジー]、[ワールド・カフェ]
