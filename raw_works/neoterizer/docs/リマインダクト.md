# リマインダクト
# 「[リマインド]を仕込む」という行動のこと。
# Remindact.
# Remind + Act.

# 背景
- 行動は重要である
- 行動というと実際に何かをすることであり、以下が挙げられる:
    - 予定や予約を入れる
    - 誰かに何かを送ったり、声をかけたりする（ボールを投げる）
    - 道具その他何か買う
    - 同上、使う
- 一方で、以下の問題も知られている
    - 即行できない問題。すぐには行動できないのでひとまず意識はしておく → 忘れる
    - やる気がない問題。振り返りでやるべきだとは理解した → やらない
- これらを軽減するためにリマインダーを仕込めば良い。安直だが Remind + Act から Remindact としてみた<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 名言
# >行動はしてますよ。リマインダーを仕込みましたから。

# 概要
- 行動として「予定を入れる」はわかりやすいが、重たい
    - 似たカテゴリーとして「リマインダーを仕込む」もある、予定よりははるかに軽い
- 重要なのは[動線]の見極め
    - たとえば私は先ほど、[週次レビュー]を経て「アイデアはつくれるが自分から提案しにいく行動が弱い」課題がわかったので、週次レビュー時に使う質問リストに「見聞きした機会すべてについて、ねじこめる提案はないか？」を追加しました。週次レビューテンプレートは私にとって動線であり、ここに書いておけば見聞きする機会が増えます。少なくとも週に一度、じっくり振り返ることがほぼ確約しました。これは、単に「来週から気をつけよ」などと思うだけでは決して成せないことです。また予定を入れるほど重たいことでもありません。しかし、意味はあります<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
<img src="https://gyazo.com/e806e5ce8ca73407797e3c463d37fd88.png" />

    - 動線（日頃から何度も通っている場所）があるはずで、そこに置いておくことで、次そこを通ったときに思い出せるようになる。思い出せたなら行動に繋がる可能性がある（少なくとも上がる）
    - 無論、動線がなければつくらねばならない（習慣と同じジャンル）し、動線があっても頻度が少なければ定着しないので、通る頻度の多い動線を探す or つくらねばならない
- 動線以外のリマインド方式でも良い
    - たとえば[スリープリマインド(Sleep Remind)](スリープリマインド(Sleep_Remind).md)でも良い
