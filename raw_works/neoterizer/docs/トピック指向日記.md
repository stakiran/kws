# メリデメ
- メリット⭕
    - トピック指向と同義
        - つまり、あるトピック（話題）に関する情報をそのページに集約できる
        - ナレッジマネジメントや言語化の観点で優れている
- デメリット❌
    - 常にトピックを意識して整理していかねばならない負担
    - ハードルが高い
        - トピックの新規や切り替え（ページ移動など）が増えるがゆえに、耐えうるツールが少ない。Cosenseなどページの作成や遷移が用意で、それなりに高速なものでなくてはならない
