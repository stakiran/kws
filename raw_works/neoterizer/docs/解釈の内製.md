# 解釈の内製
# レビュイーがレビュアー向けの丁寧な表現をつくるのではなく、レビュアーが自身で「自分に合った表現」をつくること。

# 背景
- 用語を整理
    - レビュイー：成果物を提出して見てもらう側
    - レビュアー：成果物を見る側。レビューする側
- レビュアーのパワーバランス問題は根深い
    - レビュアーの方が立場が高く、時間も少ないため、レビュアーが優勢となる
    - これはレビュイーが「レビュアーのための介護」をしなければならないとも言える
    - 無論、レビュアーがボトルネックになってしまう
        - すべてのレビュイーが、レビュアーに合わせた表現を頑張って整える羽目になる

# 概要
- パワーバランス問題は、レビュアーが「成果物の解釈」をレビュイーに委譲していると言える
- ここに内製の概念を当てはめる
    - つまり、レビュイーは解釈を人に任せるのではなく、自分でやる（解釈を内製する）
- やり方
    - レビュイーは、レビュアー向けの「介護的な加工をした情報」ではなく、必要な情報と関連する情報を持て余すことなく含める
    - レビュアーは、生成AIの翻訳能力を使って成果物を自分なりに解釈する
