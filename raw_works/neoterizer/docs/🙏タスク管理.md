# 🙏タスク管理
# タスク（やること全般）を管理すること。

# 非常に広義であり、一言で言えるものではない。
# たとえば:
- やらねばならない、やるべき、やりたい、やっておいてもいいなど全部含む
- そもそも何がタスクになるかが人によって違うし、同じ人でも状況によって違う
    - 状況（文脈）の理解が大事
    - 自分の軸も大事: [主観的重要性](主観的重要性.md)
- タスクの捉え方も人によって違う
    - 上や周囲から振ってきた「やることが確定したもの」のみを扱う人もいれば、[時間管理マトリクス]でいうと第二領域を積極的に含む人もいるし、日々の家事や食事やトイレなど仔細のルーチンを含む人もいるなど
- タスクをタスクとして扱うためには、第三者にもわかる形での言語化が必要である
    - この思想は[知的生産](知的生産.md)でも登場しますね<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- 個人のタスク管理、チームやプロジェクトのタスク管理が存在する
    - また家族など親しい二者間のタスク管理は別の色を帯びており、別ジャンルとして立ち上げた方がいい
    - [タスク管理の3P](タスク管理の3P.md)
# etc
