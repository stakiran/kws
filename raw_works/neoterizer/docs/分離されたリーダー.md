# 分離されたリーダー
# ガバナンスを、つまりは権限を手放せさせたリーダーのこと。

# 背景
- リーダーはリードに集中するべきであり、ガバナンスも抱えるべきではない
- ガバナンスはマネージャーの仕事である
- リードに専念してもらうためには、リーダーから[権限]を切り離す必要がある
    - 切り離した状態のリーダー、という概念に言及するために、この名前をつけた<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
