# Focus Orientation
# 社会生活において、何を指向するかを分類したもの。

# 概要
- Focus Orientationでは4つの分類がある
    - 1. 人(Person)
    - 2. 場(Field)
    - 3. 概念(Concept)
    - 4. 自分(Ego)
- 人はいずれか一つを強く指向していると考えられる

# 例: 所属部門内（30人）の新人歓迎会に参加した
- 人指向
    - 特定の誰か（おそらく新人）と会話したい、新人について知りたい、非言語的にも知っていきたい
    - 「🐶さんって地元広島だっけ？どういう経緯でうちに来たん？」
- 場指向
    - 新人を取り巻く場全体の動静に興味がある、誰々はどうしてるとか誰々は新人とどのような話をしているとかが気になる
    - （あー、🐱さん、やっぱり🐶さんのことチラチラ見てるわ）
- 概念指向
    - 特定の誰か（おそらく新人）、あるいはこの場からどのような概念を見い出せるだろうか
    - （ほぉ、🐶さんからはアーレントのPluralityを想起するね）
- 自分指向
    - 自分から見て、人や場や概念はどう使えるだろうか、解釈できるだろうか
    - 「🐶さんもお好み焼きは広島風を支持するよね？」
        - わかりづらいので補足すると、ここではこの自分指向の人は「広島風お好み焼きが周辺で支持されていないのが不満」であり、なんとかしたかったのを思い出して、🐶さんが広島出身であることを利用しようとしています。🐶さんが広島風を支持しているかはわかりませんが、広島出身なので支持しているとみなし、利用するための会話を始めたのです<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
        - このように自分基点で考えることを[エゴセントリック]と呼んでいます。エゴセントリックの行動原理は、通常本人から引き出さないと理解できません。しかし、自分指向であっても、空気を読んで人・場・概念指向のふりをすることはできるので、自分指向が常にそのとおりに行動するとは限りませんし、このような場に来るくらいの人物ならおそらく擬態しているでしょう<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# Person Hub
- 親密な関係を築くためには、少なくとも人指向が含まれていなければならない
    - これは人指向の者が集団内においてハブ（Hub）となるからである
<img src="https://gyazo.com/261428a3ac68b9516c0cf7e8440dae64.png" />

    - カジュアルな場面では「自分から声をかけないと人間関係は維持できない」といいますが、声をかける側の少数派はまさに人指向でしょう<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
        - たとえば5人グループの中に人指向がひとりもいないと、おそらく（会社や趣味コミュニティなど明確な場がなければ）離散しますが、人指向がいるとその人から他の人に絡みに行くため関係が持続します
        - また、そもそも、人指向がいないと中々関係が進展しません
        - もっとわかりやすいのがカップルであり、ライフステージの変化を伴うような提案（典型的には告白やプロポーズ）は通常、どちらか片方が行います。行う方は人指向である可能性が高いです。人指向は人そのものに興味があり、それが行動のインセンティブとなっています
- 人指向は「外向的」と呼ぶこともできる

# 集団において重要な指向
- 人指向
    - 唯一の外向的
    - 集団の結束や維持に不可欠
    - <a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
        - 人指向でない者が人指向のように振る舞うことでも可能であり、リーダーやビジョナリーに求められる振る舞いでもあります
- 自分指向
    - 唯一のエゴセントリック
    - 集団の変革に役立つ
    - <a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
        - 通常、自分指向は立場が弱く、単体で直接変革をおこすわけではありませんが、周囲に少なからず（良くも悪くも）影響を与えます。立場が弱いため、評価されることは通常ありません。職場を例にすると、良い場合はこの人に触発された一部メンバーが（井の中の蛙に気づいて）転職したりします。悪い場合はチーム内で敵と認定されて孤立したりいじめられたりします
        - 自分指向が他の指向を続けることは困難であるため、集団内で立場を持つ者が自分指向であることは通常ないでしょう
