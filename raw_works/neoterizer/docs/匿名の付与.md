# 匿名の付与
# [🙏組織単位](🙏組織単位.md)に匿名性を付与すること。
# つまりは、その組織単位を[クローズドな匿名](クローズドな匿名.md)にすること。

# 付与の仕方の例
- メンバー6人が集まるチャットチャンネルCがあるとする：
    - リアクションによる発動
        - スレッドに🌁（霧）リアクションをつけると、そのスレッドが匿名になる
    - 期間限定式
        - 指定した期間中、チャンネルC中の新しい発言がすべて匿名になる
- デジタルホワイトボードWがあるとする：
    - 領域指定による発動
        - この領域内に置いたオブジェクトはすべて匿名になる

# オンオフの切り替え
- 付与する場合。オフ → オンに切り替える場合
    - 既存の情報すべてを匿名化する必要がある
- 剥奪する場合。オン → オフに切り替える場合
    - 既存の情報すべての匿名性は保持した上で、以後書き込めなくする必要がある
    - または場所ごと削除する必要がある
