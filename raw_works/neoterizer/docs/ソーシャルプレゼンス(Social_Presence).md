# ソーシャルプレゼンス(Social Presence)
# プレゼンスにより社会的交流を行うという概念。

# 概要
- SNSでは主にメッセージその他情報のやりとりによって交流する
- 一方、ソーシャルプレゼンスでは、プレゼンスの共有によって間接的に交流する
- [プレゼンス]とは:
    - 自分の存在を示す何らかの情報
- プレゼンスの例:
    - チャットの在席ステータス
    - ビデオでリアルタイムに映している様子
    - n分ごとの写真など静的に撮影して共有したもの

    - 自分のGPSを共有したもの
