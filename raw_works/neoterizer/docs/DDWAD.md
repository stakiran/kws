# DDWAD
# [PFM(Personal File Management)](PFM(Personal_File_Management).md)の一種で、Desktop, Document, Work, Application, Downloadの5つのフォルダをつくるというもの。

# サムネイル:
<img src="https://gyazo.com/3d38525bec14a16e1190188f886db7a2.png" />


# 概要
- Windows用
- Desktop(デスクトップフォルダ)
    - アクティブなプロジェクトや情報は全部ここに置く
    - 整理は任意だが、机と同じで定期的に掃除してシンプルに保つ
- Download(ダウンロードフォルダ)
    - ダウンロードその他外部から入手したものはすべてここに置く
    - 整理は任意
- Work(作業フォルダ)
    - `YYYYMMDD_Name`名のフォルダをつくってアーカイブ
    - ≒Desktopで要が終わったものをアーカイブ
    - たまにしか着手しないものは、最初からここにつくって、デスクトップからはショートカットする等でも良い
- Application(アプリケーションフォルダ)
    - インストールしないタイプのツールを集める
- Document(文書フォルダ)
    - マニュアル、リファレンス、その他文書や書類全般を入れる
    - 整理は任意

# 意義
- シンプルに5つの大別を与えるのみであり、細かい管理はしない（利用者に委ねる）
    - そのためのPFMの初心者でも気軽にマイペースに導入することができます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
