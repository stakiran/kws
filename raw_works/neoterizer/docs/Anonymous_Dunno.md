# Anonymous Dunno
# 「知りません」リアクションを匿名で行うこと。

# 背景
- 主に[ビジネスチャット]において、「～～についてご存じの方いらっしゃいませんか」のような質問は多い
- 知らない場合、このような質問に答えることはできずスルーとなる
    - 🤔や👀で無理やりリアクションすることは可能ですが、統制がキツイと感じるでしょう。つまり「原則👀をつけて反応してね」のようなルールがある場合はスルーは起きませんが、そもそもそのようなルール自体が負担の高いものです。さらに、これらリアクションをつけたところで「既読」以上の意味はでません<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 概要
- 「知りません」をリアクションするためには、匿名にすればいい
    - [クローズドな匿名](クローズドな匿名.md)
    - 仮に❓️を使うとして、❓️3とある場合、3人が「知りません」と答えたことになる
        - ただし匿名なので誰が答えたかはわからない。そのチームメンバーのうち3人が答えたことだけわかる

# 実装
- ビジネスチャットはこの機能を搭載していないため、新たに実装が必要
    - TeamsやSlackなど既存の製品にそんな機構はないため、新しいビジネスチャットツールを作る必要がある
