# ダシズム(Dashism)
# 人も含めて、あらゆるものをダシにして[知的生産](知的生産.md)を行おうという立場。

# ダシにするとは？
- 辞書的には「自分の利益のために利用する」「自分の都合のいいように使う」など
- この文脈で言えば、好き勝手に取り出して、解釈して、使うこと

# hereの立場を取る者を[ダシスト(Dashist)]と呼ぶ。

# [ダシストのジレンマ]がある。
- >見せると嫌われる。見せないと偏る。
- ダシにする行為は、他者に（特に本人に）見せなければ実害はない
- しかし、知的生産である以上、成果物は他者に見せるか、見えるよう公開せねばならない
    - その成果物はダシにしてつくったものであり、本人が見れば、ダシにされたことがわかるかもしれない
    - そうでなくとも、成果物には過程もつけることがあり、この場合はもろにわかる
    - 本人が傷付く、機嫌を損ねるなどの実害が生じる
        - ダシにするとは自分勝手な加工であり、対象（および対象の主）へのリスペクトはない or あったとしても勝手な加工をしているので意味がない
