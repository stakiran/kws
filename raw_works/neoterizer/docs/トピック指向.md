# トピック指向
# 1つの情報単位には、1つの話題だけを書き込むという考え方。

# 背景
- [Cosense]では1ページ1話題で書いて、ページ間をリンクでつなげることでネットワーク的な知をつくっていく、との哲学や文化がある
- これは非常に革新的かつ有用であり、ナレッジやノートを扱う上で汎用的に使えるので、名前をつけた<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 概要
- 設けた情報単位を[* トピック]という
- トピックの作り方はシステム次第
    - Cosenseの場合、ページでつくれる
- 1トピック 1URLが良い
    - これにより言及や参照がしやすくなるし、複数の話題が混ざるのも防げる（混ざっても分離できる）
