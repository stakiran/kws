# オープン・エンジニアリング
# 情報共有をよりオープンに行い、真にオープンな組織を実現するためのエンジニアリング。

# [整備中]<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 触発
- Web2.0、特にオープンソース

# 動機
- 人類の水準を上げるため
    - 情報は非常に重要である。特に仕事とは情報の交換とみなせる
    - しかし交換の仕方の大部分は現代でもなお原始的である
        - たとえば[公然の職権乱用](公然の職権乱用.md)が行われている
    - 上げ方は色々考えられるが、端的に上げるためにはオープンの概念が良い
        - Web2.0、特にオープンソースの形で実証されている
