# 認知的貧乏
# 経済力ではなく認知資源が貧乏であるさま。

# 背景
- 現代人は認知資源が乏しい傾向にある
    - 情報過多、多忙、不健康のトリプルコンボにより、認知資源が恒常的に足りない
    - あるいは目の前の仕事や生活の分しか残っていない
- つまり「考え事に使う分」が残っていない
- 弊害:
    - 新しいことを勉強したり、試したり、[文化的認知](文化的認知.md)を行ったりといったことができない
    - 認知資源の重要性を理解しておらず、[認知的安全性(Cognitive Safety)](認知的安全性(Cognitive_Safety).md)がない → 認知資源を確保できない → 確保できないから現状を変えられず[木こりのジレンマ]に陥る、の負のスパイラル

# 概要
- 認知的貧乏の概念は、認知資源もまた貴重なリソースであることを突きつける

# 認知的貧乏から脱するには
- [労働者体験](労働者体験.md)を上げる
- [管理3.0]を採用する
- 拘束的な働き方から脱する
    - 拘束は全員の時間と場所を縛るものであり認知資源も奪います。そうではなく各個人がフルリモートフルフレックスフルアシンクをデフォルトにして働き、その進捗や結果を共有して進めていくようにします。これにより、たとえば従来は出社と会議で7時間を費やしてたものが、個人ワーク5時間とコミュニケーション2時間になったりします。このサイトでも多数の知的生産物を紹介しているので参考にしてください<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- 健康的な生活を整備する
