# デイリーライン(Daily Line)
# 日単位の動線。

# 例
- カレンダー
- デイリータスクリスト
- デイリーノート
- 日報

# 概要
- 動線とは:
    - 毎日それなりの頻度で行き来する場所
    - ここでは物理的というよりデジタル的
- 日単位の動線とは:
    - 動線の場所が毎日変わること。特に今日の日付が動線となること
    - たとえば今日が2025/10/05の場合、カレンダーだと「2025/10/05」の表示が今日の動線です。デイリータスクリストやノートの場合は「2025/10/05のノート」が動線です<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- デイリーラインとは:
    - 日単位の動線のこと
    - デイリータスクリストやデイリーノートなどを抽象化した概念としてつくりました。ラインとは動線を差します<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 意義
- 未来日に仕込めること
    - 3日後に確認したい場合、3日後のエリアにその旨を書いておけばいいのです。これによりその時が来るまで忘れることができ、認知資源の消費を抑えられます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- 他の実装を検討できること
    - タスクリストやノート以外の概念もつくれるでしょう<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
