# ラピッド・ライブ・プロトタイプ(Rapid Live Prototype)
# 雑に描いたUI上に、雑に文字入力したりコピペしたりすることで実演するプロトタイプ
# RLPと略す。

<img src="https://gyazo.com/a1fa0157984918b812595ef31fe67104.png" />

        - このアイデアも蒸留しています →[/modularnotions/キュライン]

- メインタイムラインにつぶやきが投下されていく様子と、その一部をクリップしていく様子を実演している

# やり方
- 1 雑にUI（として機能する下地）をつくります
- 2 雑にシナリオを考えます。どこに何を打ったり移したりするかの流れ
- 3 実演します
    - その場で見せてもいいですし、上記のようにGIFその他動画にしてもいいです

# 応用
- 文字入力時、自分で打つのではなくクリップボードの充填させると良いでしょう
    - [弾倉としてのクリップボード](弾倉としてのクリップボード.md)
