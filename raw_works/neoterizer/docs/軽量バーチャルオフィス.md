# 軽量バーチャルオフィス
# Light-weight virtual office.
# [Remotty]や[Teracy]のような軽量なバーチャルオフィスのこと。

# 背景
- 通常のバーチャルオフィスよりも軽量なあり方が見られるので、名前をつけて捉えた<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 原理
- [ソーシャルプレゼンス(Social Presence)](ソーシャルプレゼンス(Social_Presence).md)を使う
- コミュニケーションにつなげるための各種機能も提供する
    - 特にビジネス用途では「必要に応じたコミュニケーション」を「迅速に」開始できることが重要であり、このための各種機能をいかに提供するかがLVOの腕の見せ所
