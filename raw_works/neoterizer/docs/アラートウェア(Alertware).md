# アラートウェア(Alertware)
# 特に緊急性の高い通知を行うためだけのツール。

# 背景
- [ベルウェア(Bellware)](ベルウェア(Bellware).md)を参照
- ベルウェアだけだと緊急性の高さに対応できないため、対応するツールを別途つくる → アラートウェア

# 意義
- [BFAF](BFAF.md)
    - Before: チャットや電話を使っていた。常に通知されるため、本当に緊急だろうがそうじゃなかろうが、毎回確認しなければならなかった or 諦めてすぐの確認をやめて怠惰キャラにならねばならなかった
    - After: 本当にすぐ対応せねばならない通知はアラートウェアに限定されるので、平穏が保たれやすい
