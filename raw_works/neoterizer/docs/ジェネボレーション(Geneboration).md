# ジェネボレーション(Geneboration)
# 生成AIを用いた[🙏コラボレーション](🙏コラボレーション.md)のこと。
# Generative + Collaboration.
# 形容詞はジェネボレーティブ(Geneborative)。

# 概要
- コラボレーションとは:
    - n人のメンバーが、指定された場に、情報を出すこと
    - 出したものに手を加えたり、別の思いついたものを出したりして、広げたり深めたりすること
    - [🙏発想法的](🙏発想法的.md)は好例
- 貢献とは:
    - 場に情報を出したり、出されたものを参考にさらに出しだり、出されたもの整理したりすること
    - つまりコラボレーションとは、n人のメンバーが指定された場で貢献し合うこと、と定義できます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- ジェネボレーションとは:
    - 「メンバーに相当するレベルの貢献を行う生成AI」がいるようなコラボレーションのこと

# 例
- Fostering Innovation with Generative AI: A Study on Human-AI Collaborative Ideation and User Anonymity

# 違い
- Q: [QWINCS](QWINCS.md)製品に見られるAI連携機能はジェネボレーションですか？
    - Ans: いいえ
    - それらは人が機能を呼び出すことではじめて起動するため、ジェネボレーションではありません
    - ジェネボレーションでは、生成AIは自律的または自動的に動きます
- Q: [tokoroten/NovelDrive https://github.com/tokoroten/NovelDrive]はジェネボレーションですか？
    - Ans: はい
    - 生成AIが自動的に（たとえば指定間隔で）動いているからです
- Q: AIエージェントは現在のところコーディングその他開発作業がメインですが、コラボレーションを担当するものが出現した場合、それはジェネボレーションですか？
    - Ans: はい
    - 生成AIが自律的に動いているからです
