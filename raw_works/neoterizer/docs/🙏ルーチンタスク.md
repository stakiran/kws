# 🙏ルーチンタスク
# 特定の頻度で実施するタスクを指す。

# 別の言い方
- リピートタスク
- 定期タスク
- 繰り返しタスク
- 日課・習慣
    - 日課や習慣はルーチンタスクの一種です<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 背景
- タスクの大半は「繰り返し行うもの」であるか、繰り返し行う営みに帰着できる
    - 便宜上名前をつけたくて、ルーチンタスクを名付けています<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- 人は通常、20-40のルーチンタスクを抱えている
- しかし[忘迷怠](忘迷怠.md)が発生するため、効率的に処理出来ているとは言えない
- タスク管理を使うと、忘迷怠を防げるので、その分効率的に処理できるしQoLも上がる
    - たとえば1日2時間の時間と100の認知資源を要していた日々のルーチンタスクを、1時間と30でこなせるようになります。その分、空いた時間や認知資源を別のことに使えます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
