# ダブルノート(Double Notes)
# 1つの情報単位（ノートやページと呼ばれるもの）に、PublicとPrivateの二つの領域を設けるという概念。

# 背景
- 通常、情報単位には一つの領域がある
- 公開性を分けたい場合、情報単位ごとに分けるか、情報単位を集めた単位ごとに分けるかの二択が主流だが、やりづらい:
    - 特にPrivateで下書きしたものを、Publicに移す際がシームフル
- そこで第三のあり方を提案しました。情報単位自体に表（Public）と裏（Private）を設けて、通常は表に書いたものだけが使われる、とするのです<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 意義
- Public-Privateの行き来がシームレスになることです
