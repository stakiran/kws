# 共感トピック
# [トピック指向](トピック指向.md)の一種で、トピックとして共感してほしいテーマを掲げ、そこに共感をぶら下げる（ぶら下げてもらう）あり方。

# 例
- さらけだ荘
🌎️[ソフィ公式TikTok「さらけだ荘」開設から2ヶ月間で600万回再生を突破-ユニ・チャーム](https://www.unicharm.co.jp/ja/company/news/2023/0629-01.html)
    - 生理に関する情報発信をTikTokで行っているが、TikTokの動画一つ一つがトピックとなっており、共感した視聴者がリアクションやコメントを残している<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
    - [トピック指向](トピック指向.md)ではページなどテキストコンテンツを想定していたが、現代はショート動画の時代でもあり、特に若者はノウハウもショート動画で提供・消費している。トピックとして動画を用いるのはなるほどと思った<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
