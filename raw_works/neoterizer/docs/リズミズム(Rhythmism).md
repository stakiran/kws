# リズミズム(Rhythmism)
# リズミカルに過ごしたり、捉えたりすることに重きを置くこと。

# 背景
- そもそも人間は生理的にはリズミカルである
    - [サーカディアンリズム]、[ウルトラディアンリズム]など
    - 昼夜など環境そのものがリズミカルであることも関係するだろう
        - もっと言えば物理の根幹を成す波自体が、リズミカルである
    - 生活リズムを整えることが健康に繋がるのも、リズムを踏襲した方が負担が少ないからだろう
- 認知的にもリズミカルな方が易しい
    - [ハイプ・サイクル]やライフサイクルなど、循環的なリズムで捉えることはよくある
- 精神的にも優しい
    - リズミカルということは周期性があり、ある種一定であり、読めるということである
    - アトランダムよりも優しい
    - また音楽やゲームはしばしば「リズムの演出」から成っており、リズムは人を楽しませ、奮い立たせ、やる気を起こす作用を持つ
- 体力的にも最適解
    - 多くの場合、一定リズムを保った方が最も保つ
    - マラソンはわかりやすい
