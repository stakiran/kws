# またビジネスでは「知的生産性」として言及されることが多い。
- <a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
    - 多少強引にまとめると、革新効率（Innovation Effiency）とでも言えるでしょうか
    - 計測により管理的に効率を上げるよりも、効率の高い仕組みやプロセスを探求して再現性や偶発性を上げる方向への舵取りが目立ちます
🌎️[平成23年度知的生産性研究委員会](https://www.mlit.go.jp/jutakukentiku/house/torikumi/chiteki/H23/chiteki_gaiyo.html)
    - >経済成長を遂げていためには、人的資源の活用、すなわち知的生産性
    - >知的活動の階層
        - >情報処理: 知識情報の定型処理
        - >知識処理: 知識情報の調査・加工、知的価値向上
        - >知識創造: 価値創造、イノベーション
🌎️[知的生産性研究所｜内田洋行](https://office.uchida.co.jp/chiken/)
    - Knowledge Collaborationがあてられています
🌎️[知的生産性とは？｜ オフィスレイアウト用語集（イラスト付き）](https://www.megasoft.co.jp/3dod/glossary/intellectual_produbtivity.php)
    - >知的生産性とは、独創的・画期的なアイデアやビジネスモデルなど、事業・収益の基盤となる知的成果を生み出す効率のこと。
