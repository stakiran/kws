# 🙏言い訳オブジェクト
# 同座中の気まずさから逃避するために使えるもの。
# 別の言い方をすると、会話することなく同座を持続させるのに重宝するもの。

# 例
- 会食中の食事
    - 食事行為に勤しむことで逃避できる
- スマホ
    - 状況次第では失礼にもあたるが「スマホいじってます」の体裁は現代では使いやすい
- ながら見している仕事場
