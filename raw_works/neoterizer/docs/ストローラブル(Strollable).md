# ストローラブル(Strollable)
# 「散歩可能な」の意。割り込みも目的もない30分の散歩を1-strollと定義し、月あたり何strollかを測る。

# 概要
- ストロールとは:
    - 「無目的かつ割り込みもない、30分の[散歩]」を指す単位
    - 以下はストロールではありません
        - 用事例1。通勤、買い物、食事、人と会うなど何らかの用事のために移動している
        - 用事例2。散歩ではあるし、スマホもいじらないが、近所の子供やペットその他動物などが目的になっている
        - 割り込み例1。散歩ではあるがスマホをいじっている or 普段いじっていないが連絡可能にはなっており、連絡が来たらいじる
        - 割り込み例2。パートナーや友人など複数人で散歩する
        - 割り込み例3。犬その他ペットと散歩する
- ストローラブルとは:
    - ある日または（半日や午前などまとまった）時間帯において、ストロールを差し込めること
- ストローラビリティ(Strollability)とは:
    - ストロールを週または月あたり何回差し込めたかという指標
    - このページでは週を扱うことにします<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- ストローラビリティによりライフスタイルやワークスタイルを推し量ることができる
    - 1. Expect（期待）。あなたがほしいストローラビリティ
    - 2. Actual（実際）。あなたの実際のストローラビリティ
    - 両者のギャップをはかり、縮めていくことで、あなた自身に合ったスタイルに近づかせることができ、QoLや[QoW](QoW.md)が上がる

# 目安
- ゼロ・ストローラビリティは珍しくない
    - ストローラビリティが0（特に実際も 0 で期待も 0 と両方ゼロ）であることを指す
- フル・ストローラビリティは珍しい
    - 少なくとも期待が7以上、つまり1日1回は休日も含めてストロールを入れたいことを示す
    - 現代は慌ただしく、週のすべてにおいてストロールを差し込むことは通常難しいですし、望みません。望むということはそれだけでレアキャラだと思います<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- ヘビー・ストローラビリティも珍しい
    - 期待も実際が14以上であることを指す
    - つまり1日1時間以上のストロールをしています。これはストロールそのものに価値を見出しており、たとえば雨の日や夏季冬季でも例外なくやろうとすることが多いと思います。性分としてはフル・ストーラビリティと変わりません。どちらかと言えば、フルに「ストロールが趣味」が加わったものと考えると、理解しやすいはずです<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 応用
- 複数人同伴の散歩であってもストロールにすることは可能
    - これを[* パーティー・ストロール]という
    - ただし要件は厳しく、以下をすべて満たす必要がある。
        - 1. 散歩するコースと所要時間がおおよそ決まっていること
        - 2. 散歩中、一言も一切会話しないこと
        - 3. 散歩中、はぐれることも許容すること。また無理に合流したりペースを合わせたりしないこと
    - <a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>これらを満たさない場合、十中八九どちらかが会話を始めるか、またはどちらに話しかけられる形で会話が発生します。この概念は、パートナーとどうしても一緒に散歩したいというお客様のニーズを満たすために開発したものですが、正直難しいと思います。私は「いや、ひとりで散歩してください」「それができないならゼロ・ストーラビリティです」と言うことにしています
