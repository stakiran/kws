# Weekly Menu Count
# メニュー（1食分の献立）が週に何種類あるかを表す指標。
# 略称はWMC。

# 例1:
- 1日3食で、毎日違うものを食べるとしたら、21wmcです
# 例2:
- 朝食はBASE BREAD、昼食は炊き込みご飯、夕食はチェーン店Aのメニュー2つ or Bのメニュー3つ or Cのメニュー1つのいずれかかが選ぶとしたら、1+1+(2+3+1)=8wmcです
    - ただし「今週はCを気に入ってずっとそれを食べた」のなら3wmcになります
# 例3:
- 1日n食で不定だが、食べるものと通う店は決まっており計13メニューである、しかし間食が多くこの分もメニューに含めると数十どころではなさそうであるという場合、13wmcです
    - つまり間食は含めません

# 定義
- メニューとは「それだけを摂り続けても半年以上普段の調子とパフォーマンスを持続できるもの」を指す
    - ですので間食は通常は含まれないはずです<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 概要
- パートナー（生活でも仕事でも）との相性を測るのに使える
    - ライフスタイルがマッチすると相性が良くなる（少なくとも悪くなる主因のそれがない＝悪くなりにくい）が、ライフスタイルはWMCに表れると考えられるため
- WMCの真価が発揮されるのは旅行時や引越し後
    - もしこれらのときにWMCが高くなったとしたら、その人は元来高い側の人間だと考えられる
    - 逆にWMCが元来低い人は、これらの場合でもすぐにメニューを確定させて、WMCが早く（低めの値に）収束する
    - パートナーとの相性をはかる意味でも、まさに使えるでしょう。たとえば会社で一週間の箱根合宿に来ているときに、WMCが低い人は栄養や美味しさなどを鑑みてバランスの良いラインのメニューを固定するでしょう。一方、WMCが高い人はそんなのあんまりだ、と反抗し、いろんなメニューを楽しむ（もちろんそのための時間も確保）するでしょう。両者は十中八九衝突します。ですので通常は食事時間は別行動にするでしょう。もしWMCが近い場合、そこも合うことができるので、食事時間もともにすごせて、さらに濃厚に過ごせます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 開発秘話
- <a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>パートナーの相性を推し量るための端的な指標がほしいとのリクエストを受けて、つくったのがこれです
    - 話のタネになりそうで、かつ共有もしやすそうなものを選べたと思います。実際に相性判断にどの程度有効かは要検証ですが、楽しく計算したり会話したりできると思います。みなさんもぜひお試しください。
    - 有効性次第ですが、転職の面接で聞いてみるのもアリかもしれません。まずWMCの概念の説明から始めて、聞いてくれない場合は親身じゃない組織なのでお断りした方がいいでしょう。本題はここからで、聞いてくれた場合は、WMCが合うかを見ましょう。あなたが多いが相手が少ない場合、あなたのワークライフバランスはおそらく削られます。逆にあなたが低くて相手が多い場合は、相手はそれなりにワークライフバランスを重視するか、食事込みの交流を行うので、あなたとしては消化不良またはノイジーかもしれません。
