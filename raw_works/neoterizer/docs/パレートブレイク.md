# パレートブレイク
# [パレートの法則]に見られるような「少ない強者が全体を支えている」現象から脱すること。

# 概要
- 多様性の問題と考えられる
    - つまり[画一主義(Uniformism)](画一主義(Uniformism).md)など単一的なあり方しかないがゆえに、それに合わない者がパフォーマンスを発揮しきれずに強者になれないという構図だと捉えることができる
    - あり方をn通りに増やすことで、強者になれる率も増やせる。これは多様性を確保することとと同義である
- 注力対象をいたずらに散らすことではないことに注意してください。ビジネス上、重点となる対象があるのは多くの場合、妥当です。ただ、画一主義でも挙げたとおり、[🙏仕事術](🙏仕事術.md)の観点で単一にしたがることを指摘しています。たとえば一律的に全社員に特定頻度の出社を課す、といったことです。その単一性がボトルネックとなっているのです<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- 多様性を確保するには:
