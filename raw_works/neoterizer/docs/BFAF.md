# BFAF
# Before/Afterの略で、行指向的に両者を端的に書くためのフォーマット

# 例
- BFAF
    - BF: ここにbeforeを書く
    - AF: ここにafterを書く
# 注釈もつけてます。より親切にする場合は「Before: 」のように書きます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 例2
- BFAF
    - ここにbeforeを書く
    - ここにafterを書く
# 一行目がbeforeです<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 例3:
- BFAF: (ここにbeforeを書く) / (ここにafterを書く)
