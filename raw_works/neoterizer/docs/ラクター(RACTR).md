# ラクター(RACTR)
# 以下の性質を持つ運動のこと。
# 不器用でも末永く運動を楽しめるあり方。

# RACTR
- Rhythmical
    - リズミカルに動かすこと
- Alternately
    - 交互に動かすこと
- Continuously
    - 瞬発的でないこと
    - たとえば1分以上続けられること
- Toolless
    - 道具を使わないこと
- Repeatly
    - 同じ動作を反復すること

# 実践
- 1 何らかの「動き」を選ぶ
- 2 何らかの「音楽」を選ぶ
- 3 音楽を流しながら、それに合わせて動きをする

# 実践例
- ランニングマン縛り
    - シャッフルダンスの動きの一つである[ランニングマン]のみを用いて、踊ったりトレーニングをしたりする
- リズム階段
    - 音楽に合わせて、階段を一歩ずつ昇降する
    - 環境が許せば非常階段が使えるし、屋外の長い階段でも良い
        - [釈迦院御坂遊歩道]とまでは言わないでも、階段が長くて人が少ないスポットはあるはずだ
- スピードステップ
    - 音楽に合わせて、8分間隔または16分間隔（連打の間隔）で交互に足踏みする
    - 音ゲー用語でいう地団駄をずっとやる

# 音楽と出会うには
- おそらく[音ゲー]がてっとり早い
    - 個人的には一番おすすめ。音に合わせて動くという営みは、縁がありそうで意外とない。慣れないとスタートラインに立つことすらできない。音ゲーは役に立つ。一方で、音ゲーの時点で「合わない」と感じたら、おそらくラクターも合わないので撤退すれば良い<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- もちろん、映像作品の音楽、なにかのBGMなど何でも良い
- 自分がビビッと来た音楽に、素直に手を出してみる
