# Wavelike
# プラスとマイナスの振れ幅が大きいさま。波の性質を持つさま。
# ウェーブライク。

# 例
- 人付き合い（特に親しい人）
    - 満足感や便宜などプラスも大きいが、衝突や喧嘩などマイナスも起こる
- 旅行
    - 日常では得られない体験や出会いはプラス、準備の手間や慣れない土地でのストレスはマイナス
- 一般的な意味での人生
    - 山もあれば谷もあるといわれる

# 意義
- ウェーブライクが唯一のあり方ではないと知れること
    - 例: [ソロイズ(Soloize)](ソロイズ(Soloize).md)。人付き合いというウェーブライクな営みを意図的に回避している
