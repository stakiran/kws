# 差し出しパラダイム
# 「始めるために何を差し出すか」のパラダイム。

# 概要
- Meet/会え
    - 私と始めたいなら、私と会え
    - 会いたがり
- Read/読め
    - 私と始めたいなら、私のを読め
    - 読ませたがり
- Chat/喋れ
    - 私と始めたいなら、私に喋れ
    - 聞きたがり

# 使い方
- 「入社や入団などジョインの審査において、一つだけ使うとしたらどれを使う？」を尋ねる
- これによりその人の差し出しパラダイムがわかります<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
    - Meeterは会いたがります。[📝関係ベース](📝関係ベース.md)を地で行きます。文化にとらわれていることも意味しており、多様性を軽視します
    - Readerは読ませたがります。読んだことを前提に濃密なやりとりをしようとします。言語を重視し、非言語を軽視します
    - Chatterは聞きたがります。権限委譲する気マンマンですが、目は肥えていますし厳しく見ます
