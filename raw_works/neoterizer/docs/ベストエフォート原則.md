# ベストエフォート原則
# [🔗知的生産の原則](🔗知的生産の原則.md)の一つで、[🙏ギャランティ](🙏ギャランティ.md)よりも[ベストエフォート]を拠り所にすること。

# 概要
- 知的生産は、実用的には問題解決ではなく問題設定に寄与する
    - やると決めたことを確実にこなすフェーズではなく、その前段階の何をどうやるかを決めるフェーズの営みである
    - ❌こなすための営み
    - ⭕決めるための営み

# 例: [ABCing](ABCing.md)
- 知的生産では、仮にABCの3つを定めて、これに当てはまるように整理していくことがよくある
    - ABCingと呼んでいる<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- ギャランティ的な価値観だと網羅性とその証明がないため信用できないが、知的生産的にはそれなりの努力でそれなりの情報整理ができているという点で有益である
    - 100点よりも60点、といった比喩はよくあるが、まさにその一つである
    - 60点、あるいはもっと低いかもしれないが、概念にして提出することで利活用や議論が可能となる
