# ウォーリー(Worry)
# 具体的な心配事の、その根本原因を指す。

# 概要
- ウォーリーは以下を満たす
    - 1. 中長期的に付き合い続けることが半ば確定していること
    - 2. 一週間に一度は相対すること
    - 3. ウォーリーに影響を与えることができないこと
        - ウォーリーの解消（破壊や追放など「なくす」）が難しい
        - ウォーリーの部分的改善が難しい
        - ウォーリーへの提案や進言自体も難しい。やっても刺さらないか、スルーされるか、あるいは単に攻撃し返されてこちらの身が危うくなる
- 例
    - チームに攻撃的な人がいる
    - 無視できないパワーの強い顧客の側に被虐癖の人がいる
    - ハラスメント気質のパートナーと暮らしている
    - 会社や部門に多様性がなく、自身のマイノリティに配慮してもらえない

# 開発秘話
- 名前の由来は、ウォーリーを探せから取ってきました
    - 「探す」ということです。ウォーリーを血眼になって探すように、あなたが現在抱える具体的な心配事の根本原因を、血眼になって探すのです。探す必要があるのです。このニュアンスをわかりやすく伝えるために、ウォーリーを探せが使えるのではないかと考えました
