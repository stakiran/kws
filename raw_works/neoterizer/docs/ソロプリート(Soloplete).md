# ソロプリート(Soloplete)
# 「単独完結的な」という意味の形容詞。または「単独完結性」という意味の名詞。特に初心者であってもひとりで作業や仕事が完結できるようになれること。
# Solo + Complete.

# 概要
- 必要なリソースをひとりで入手しきれることを指す
    - たとえば知識が整理されていて読める、必要な手続きが申請制になっていて使える、新人であっても問題なく使える（[Whoism](Whoism.md)的な差別を受けない）etc
- ソロプリートは自然に発生することはない
    - 何もしなければナレッジと手続きは必ず属人化する。それを知る人にいかに頼るかの世界となる。つまりボトルネックであり、政治が生じる
- ソロプリートであるからといって、実際に完結できることは意味しない
    - ソロプリートは、[* ひとりで完結できるようになれる]ことを示す。もしソロプリートであっても、ナレッジが多く手続きが煩雑であれば「マスターするのに一年かかります」なんてこともありえる。ただ、ソロプリートであるならば、ひとりでもできるようになる（時間はかかるかもしれないし当人のセンスも必要かもしれないが）とは言える
