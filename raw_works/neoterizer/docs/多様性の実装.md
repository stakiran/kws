# 多様性の実装
# [多様性]として扱う要素を定義すること。

# 概要
- 枠組みと要素がある
    - 例: [クロノ・ダイバーシティ(Chrono Diversity)](クロノ・ダイバーシティ(Chrono_Diversity).md)の場合
        - 枠組み: クロノ・ダイバーシティ
        - 要素: 朝型、夜型、覚醒が早い、覚醒が遅い
- 実装として行うこと
    - 1 言語化する
        - 枠組みと要素を両方言語化する
    - 2 言語化した要素を、[スタンダーフ(Standarf)](スタンダーフ(Standarf).md)(標準的なメンバー)として採用する
        - [🙏仕事術](🙏仕事術.md)の駆使
        - 制度への組み込み
