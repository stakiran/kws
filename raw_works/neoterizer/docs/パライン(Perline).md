# パライン(Perline)
# 各行は最初に書き込んだ人の陣地となり、他の人は書き込めなくなるというあり方。
# あるようなそのような行のこと。
# Personal Line.

# 背景
- 1 [QWINCS](QWINCS.md)におけるWikiやNoteなど、同時編集可能なテキストエリアの各行は誰でも書き込める
    - だからこそ空気の読み合いが発生し、現実と同様、思っているほどコラボレーションが加速しない
- 2 一方、[Cosense]ではアイコンをぶら下げることで各自の陣地を主張できるが、これはこれで陣地が乱立されるだけであり、その先の意思決定に進みづらい
- この膠着を打破するためのソリューションとして、行単位を強制的に個人の持ち物にする発想を採用した。それがパラインである<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
