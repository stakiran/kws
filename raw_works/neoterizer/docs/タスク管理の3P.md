# タスク管理の3P
# タスク管理を個人(Personal)、チームや組織(Project)、二者間(Partner)の3つに大別したもの。

- 個人タスク管理
    - 自分のタスクを自分で管理する
    - ある仕事限定でもいいが、人生全体に広げて扱うことが多い
    - ビジネス書や自己啓発ではよく見られる<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- チームタスク管理
    - ビジネスにおいてはこちらがよく使われる現象
    - 現代は実質プロジェクトだろうからプロジェクトの名前をつけた<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
    - プロジェクトマネジメントも部分的にかする
- パートナータスク管理
    - 主な想定は夫婦だが、二者間パートナーには個人にもチームにも当てはまらない性質がある
    - ジャンルとして分けないといけない<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
