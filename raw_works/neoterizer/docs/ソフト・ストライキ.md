# ソフト・ストライキ
# [ワークライフバランス]が満たされてないことを理由に、ソフトに反発すること。

# 背景
- [静かな退職]は日本に限らず、カウンターカルチャーとして広まっている
- 本質を履き違えている人が多いのです。わかりやすく伝えるために、ストライキの言葉を持ち出して、これのソフトなバージョンだぞと表現することにしました。また拡張性も持たせていますので、静かな退職以外にも当てはめることができます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 例
- 静かな退職


# 詳細
- いわゆる[ビロンギング]（少し古い言い方では従業員エンゲージント）の調査結果からも浮き彫りになると思います。ソフト・ストライカーは帰属意識が薄く、典型質問「自社で働くことを知人に勧めますか？」もネガティブでしょう<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
