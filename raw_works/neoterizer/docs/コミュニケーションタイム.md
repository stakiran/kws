# コミュニケーションタイム
# [コミュニケーションの注入](コミュニケーションの注入.md)における概念で、コミュニケーションを行うための時間帯を指す。

# 概要
- 「コミュニケーション」として行えることは様々存在する
    - これを[* アクティビティ]という
- アクティビティの例:
    - 雑談
    - [ゲームタイム](ゲームタイム.md)
        - マイクラやFPSなどPCゲーム
        - 据え置き機や携帯ゲーム機によるゲーム
    - ボードゲーム
    - 散歩
    - 体操
    - 読書
    - 映画鑑賞
- コミュニケーションタイムの目的
    - 1. [📝グルーミング](📝グルーミング.md)を充足するため。親睦を深める行為は、食欲と同様、摂取しないと苦しい
    - 2. メンバーのことを非言語的に知るため。アクティビティを通じて、その人のことを非言語的に知ることができる

# コミュニケーションのデザイン
- どんなアクティビティを用意するのか、また社員やチームに課すのかといったことに正解はない
- 手探りで進めてもいいが、おそらくかんたんには最適化には至れないだろう
- デザイナーなる役割が必要な程度には、高度な営みである

# >コミュニケーションタイム・デザイナー（CTD）なる専門職が登場するだろう。この役割は、人や組織をどう設計し動いてもらうかを考える概念設計力と、実際のメンバーに働きかけて動かす社交実践力の双方が求められる。非常にハイレベルで、引っ張りだことなるだろう。GAFAMを始めとするテック企業の知的生産物を待つ必要は、もはやない。CTD がいれば、自らつくりだすことができるのだから。これは、そう――コミュニケーションの内製だ。
