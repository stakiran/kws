# オープンニューロ(Open Neuro)
# [ニューロ]つまりは発達障害持ちであることを公言すること。

# 背景
- その昔、いわゆるオタクは迫害の対象だった
- そんな時代からの転換期において、自らオタクであると名乗ることを[オープンオタク]と呼んだ
- 同じことがニューロにも言えるようになるのでは？なった方がいいのでは、と思ってつくった<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 公言のスタンス
- 積極的
- 必要に応じて

# 公言のニュアンス
- たとえば以下のシチュエーションで公言できる
    - 転職面談、案件の顔合わせ、その他自己紹介時
    - 趣味仲間や友人との雑談
