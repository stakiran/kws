# ココリマインド
# [コリマインド(Coremind)](コリマインド(Coremind).md)を協働的に活用すること。
# Co-Coremind.

# 概要
- コリマインドは個人で使うものであったが、チームにも拡張できる
    - 具体的には、[🙏組織単位](🙏組織単位.md)にコリマインド機能を導入する
        - すると、たとえば🐶さんが🐱さんや🐑さんにコリマインドを設定できます。もちろんコリマインドが動作して、しかるべきタイミングで🐱さんや🐑さんにコリマインドされます。これは、単に会議の場で「🐱さんは～～をお願いします」と言ったり、議事録にタスクとして書いたりしておくよりもはるかに効果的です<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
    - 概念的には、タスク管理におけるタスクのかわりにコリマインドを使うと考えれば良い
        - あるいはOutlookなどグループスケジューラーを使って他人の予定に勝手につっこんで調整したりするが、そのコリマインド版だと思えば良い
- 設置と調整の分業
    - 🐶さんが🐱さんにコリマインドを設置するが、以後は🐱さん自身が調整できる
    - これにより「やり忘れ」をなくしつつ、実際の塩梅は本人に任せられるというバランスを取れる
