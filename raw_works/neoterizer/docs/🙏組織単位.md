# 🙏組織単位
# 何らかのシステム上で表現された、特定の組織を指す「場所」のこと。

# 意図
- [クローズドな匿名](クローズドな匿名.md)などを議論するために、便宜上定めました<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- 知的生産では、特定のシステム上につくられた「組織（を指す単位）」を扱いたいことがしばしばあります
    - しかし、これを指す言葉がありませんので、定めました
- 組織単位に何かを適用するとは、その組織を表現した「何らかの場所」に対して何かを適用する、ということです
    - と、このような表現が可能になります
    - 知的生産では組織そのもの（人そのもの）よりも、それを表現した何かの方を操作するので、その何かの方を指す名前があると便利なのです

# 例
# あるメンバー6人のチームTがいるとした場合、組織単位はたとえば以下:
- 6人のみが入っているSlackのチャットチャンネル1
- 同上、チャンネル2
- 同上、Teamsのチャネル1
