# クロノ・ダイバーシティ(Chrono Diversity)
# 朝型・夜型、覚醒が早い・遅いの計4パターンのあり方があるということ。
# 特に多様性として実装すること。

# 概要

| 朝型 | 夜型 |
| --- | --- |
| 覚醒が早い | 1 朝起きて朝活動 | 2 昼起きて昼活動 |
| 覚醒が遅い | 3 朝起きて昼活動 | 4 昼起きて夜活動 |

- 朝型夜型は[朝型・夜型質問紙]などで調べられる
- 覚醒については、まだ確たる研究はなさそうだが、寝起きでガッツリ食事できるかどうかが目安になる
    - できない場合は遅いか、少なくとも早くはない
    - できる場合は早いか、少なくとも遅くはない
- 便宜上、4パターンで分けているが、実際はグラデーションである
    - 朝型でも夜型でもない、いわば昼型くらいのある
    - 覚醒も早すぎないが、遅すぎもしない塩梅がある
