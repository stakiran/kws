# リアクションセット
# スタンプベースのコミュニケーションを促進するための、簡潔なスタンプのセットのこと。

# 背景
- [スタンプ]を用いたリアクションには、非同期が難しいとされている非言語コミュニケーションをある程度代替できるポテンシャルがある
- しかし現状十分に活用されていない
    - スタンプが多すぎて使う（特に選ぶ）コストが高いからだ
- この問題を軽減するためには、使うスタンプを大胆に絞れば良い

# 設計のポイント
- ひらがなの畳語リアクションを使う
    - 畳語（じょうご）とは同じ単語や単位を繰り返してつくった言葉で、「ああ」「うぅ……」「ええー」も含む
    - 長音や三点リーダーも使うことで、微妙な機微も表現できる
        - たとえば「ああ」と「ああ？」と「あぁ？」と「あぁ……」は全部違う
- テキストに絵文字を当てはめる、という形でつくる
- 攻撃性の強い、あるいはそうと解釈できるものはなるべく避ける
    - たとえば「は？」や「はぁ」はイマイチでしょう。「はぁ……」はある程度はマシですが、悲壮感や落胆の強さをどう読まれるかでばらつきが生じます。上記のはひふへほでも「は？」を使っており、あまり良い出来とは言えません<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
