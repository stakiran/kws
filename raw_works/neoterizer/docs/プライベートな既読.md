# プライベートな既読
# グループチャットやチャンネルなど複数人が見れている場所であっても、既読状況を投稿者本人にのみ見えるようにすること。

# 概要
- [既読エンジニアリング](既読エンジニアリング.md)の一種で、既読レベルの間を取れる
    - lv0: 既読を扱わない
    - lv1: 既読を扱うが、開放性も抑えている ★ここを取れる
    - lv2: 既読を扱うし、グループ内で開放されている
    - ビジネスにおいて既読は便利な機能です。いわばリアクションをシームレスにしたとも言えます。手動でわざわざつけなければならないのと、自動でつくのとでは、雲泥の差があります。しかし、既読スルーの言葉があるように、既読はしばしば不和を生じさせます。その一端は公開範囲が広いことにあります。そこで既読の表示範囲を絞ることで、この不和を軽減します<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
