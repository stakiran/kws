# Slack, Solo, Stress
# [労働者体験](労働者体験.md)の3要素。

# [スラック(Slack)](スラック(Slack).md)
- 時間的・精神的・認知的余裕のこと
- 別の言い方: 業務時間中に誰からも割り込まれることなく、かつ疲弊や動揺もしてなくて思考を行えるような時間のこと

# ソロ(Solo)
- ひとりでいること。肯定的孤独。
- 特に誰にも邪魔されることなく、また自ら脱線することもなく、個人的で深い営みに集中できる状態

# ストレス(Stress)
- 適度な圧力
- 前提:
    - [労働者体験](労働者体験.md)の文脈では何らかの仕事があり、仕事に直接的・間接的に関する何かをしなければならない
- 前提を満たすには、仕事に関する適度な圧力が必要である
    - 圧力がなさすぎる場合、進展が生まれない。いたずらに怠けてしまったり、全く関係がないことをして疲弊してしまったりする（ネットサーフィンやゲームなど）
    - 圧力が高すぎる場合、創造的に冒険したり余裕したり試行したりする余裕もなく、「終わらせるために手段を選ばず動く」インセンティブに支配されてしまう
- 適度な圧力は人によって違う
    - 傍から見るとワーカホリックと言いたくなるようなマルチタスクが適切かもしれないし、逆に「よくもなぁそんな[退屈]な過ごしかたができるよな」と感心するほど手持ち無沙汰に過ごすのが適切かもしれない
- 圧力には種類がある
    - 環境圧力。自宅、カフェ、オフィスはおそらくこの順で圧力が低い
        - もちろん人によって自宅は家族がいて圧力が高い，などもありえる
    - 文脈圧力。締切が近いと高い、声大きい人の依頼だと高い、失敗や未達時の損害が大きいと高い etc
