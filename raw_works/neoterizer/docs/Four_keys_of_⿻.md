# Four keys
- 多様性 Diveristy
    - 均質であるよりも多様である方がコラボレーションが加速する
    - しかし要素が異なるとわかり合うのは難しいし、衝突や紛争による分断も起きがち。これを技術で克服する
    - たとえば生成AIを使えば柔軟な翻訳ができるので、世界中の誰もが意思疎通を交わせる。また環境を擬人化して状態を喋らせることで感情移入してもらい、環境問題について考えてもらいやすくなる（かもしれない）といった応用もいくらでも可能である
- 持続性 Sustainability
    - サステナビリティの概念は改めて言うまでもあるまい。地球環境がまさにそうであるように、好き勝手に発展させていると土壌そのものが侵されていく
    - たとえば生成AIについても、むやみにフル活用して忙しなく動いてしまうと、[三熟](三熟.md)による深い成果物を出せない。生成AIはデータの学習により成り立っているものであり、良質なデータ（知識含む）を食わせることも重要である。良質なデータをつくれなくなっては元も子もない
- 連接性 Network
    - 階層構造には色々と限界がある。ネットワーク構造であればこの限界を突破できるし、人間関係や脳の仕組みからもわかるように、自然でわかりやすい仕組みでもある
    - インターネットは⿻の良い例である
- 超加性 Super Modularity
    - 「全体」が単なる「部分の総和」よりも大きいことをスーパーモジュラーという。比喩的には、1+1は2かもしれないが、1+1+……1 が100人、1000人、万人、それ以上となれば、ただの足し算を超えたパフォーマンスと現象を引き起こせる。実際、世の中は複雑系であり、この性質があることが示されている
    - このスーパーモジュラーを、対人関係や組織またその連携においても積極的に狙っていく
