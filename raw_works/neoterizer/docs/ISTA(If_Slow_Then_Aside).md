# ISTA(If Slow Then Aside)
# 遅い人は寄りなさい、という原則。
# 読み方は「イスタ」

# 概要
- 低水準への[画一主義(Uniformism)](画一主義(Uniformism).md)はよく起こるが、そのうち遅い者への画一化を扱う
    - 例1:
        - 追い越せない幅の道なのに歩くのが遅い、自転車が遅い
        - 同上、並んで歩いていて道を塞いでいる
        - 二列が可能な幅のエスカレーターにおいて、片側を空けずに立ち止まって通せんぼする
    - 例2:
        - 講義や演習において、最後の未完了者が終わるまで全員を待たせる
- これを防ぐためには、遅い者が譲るのが良い
    - 例1のように、遅い者が譲るアクションをすることで回避できるケース
    - 例2のように、その場をコントロールしている者が（制限時間を設けたり習熟者別にコースを分けたりなどして）回避できるケース
        - この場合、コントロールする者が遅い者を思いやることで生じていることが多いため、遅い者側から提案することで（脱・画一化に）気づかせることができます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- 譲るとは:
    - 自分と同じペースを他者に強要させないための働きかけを行うこと
