# エゴアロマトリックス(Ego-Allo Matrix)
# [エゴセントリック]と[アロセントリック]に從って、人のあり方を分類したもの。
# 単にエゴアロで捉えるのではなく、気質と振る舞いで分けたことにより、高解像度な捉え方が可能となる。


| アロ気質 | エゴ気質 |
| --- | --- |
| エゴ的に振る舞う | 1 傲慢者  | 2 エゴ |
| エゴ的にもアロ的にも振る舞う | 3 一般人 | 4 擬態者 |


- 1 傲慢者
    - アロ気質だが、エゴに振る舞う者
    - 俺様などナルシスト、王様など権力者、亭主関白など差別的な文化（の優位な方）などで起こる
- 2 エゴ
    - エゴ気質でエゴに振る舞い者
    - [ASD]など
- 3 一般人
    - アロ気質で、エゴとアロを使い分けられる者
    - 通常はここ
    - ただしエゴとアロの使い分けは状況次第、人次第
- 4 擬態者
    - エゴ気質だが、アロの使い分けもできる者
    - IQの高いASDや社会人経験の長いASDなど、才能や経験でカバーすることで至れることがある

# 対処
- 傲慢者は追放するか矯正せよ
- エゴは、多様性としてエゴが許されるあり方をつくってそこに行ってもらえ
- 一般人は言う事はない
- 擬態者はも言う事はないか、擬態が甘い場合は指摘して改善させよ
