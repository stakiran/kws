# このサイトの読み方
# ページ名
- 「無印」と絵文字付きを使います

| (表記) | (解説) | (例) |
| --- | --- | --- |
| 無印 | 私がつくったもの | [知的生産] |
| 運用系のページ | [このサイトについて], [settings] |
| 既存の言葉 | [Cosense] |
| 📝付き | 既存または他者の言葉 | [📝知的生産] |
| 🙏付き | 既存の言葉だが独自定義 | [🙏組織単位] |
| 📖付き | 書名 | [📖孤独に生きよ 逃げるが勝ちの思考] |
| 🔗付き | リンク集 | [🔗3rd Paradigm] |
| ❌付き | 廃止 | [❌ハイコンテクスト] |
| 🏮付き | お祭り会場 | [🏮⿻本] |
| ⚒️付き | ツールその他道具 | [⚒️pgpt] |

    - 既存の言葉は基本的に解説しませんので、各自調べてください
        - どう使われているかを辿ることはできます → [ゴーストページ](ゴーストページ.md)
    - 便宜上、独自な定義で上書きすることがあり、その場合は🙏をつけています
        - 絵文字のとおり、すみませんって気持ちです
- [ページ名に英語がついたりつかなかったりする理由](ページ名に英語がついたりつかなかったりする理由.md)
