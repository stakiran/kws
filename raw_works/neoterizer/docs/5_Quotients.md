# 5 Quotients
# 根本的な能力を推し量るための指標として、<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>が提唱したいもの。

- [IQ(Intelligence  Quotient)]
    - 知能指数
    - 言語理解、知覚推理、ワーキングメモリー、処理速度
    - 知的障害だと下がる
    - 既存です。[WAIS-IV]など<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- PQ(Physical Quotient)
    - 体力指数
    - 運動神経、燃費、相対筋力
    - 身体障害だと下がる
    - 既存で、子供の身体的健康を測る指標のようです。私の想定は違います<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- BQ(Busy Quotient)
    - 多動指数
    - 創造性、[ファンタジアビリティ(Fantasiability)](ファンタジアビリティ(Fantasiability).md)
    - ADHDやハイパーファンタジアだと上がる
    - まだありません<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- SQ(Social Quotient)
    - 社交指数
    - 文化の理解、非言語の理解、[アロセントリック]性
    - ASDだと下がる
    - 既存で、EQを開発したゴールマンの第三の指標だそうです。私の想定は違います<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- DQ(Delicateness Quotient)
    - 感情の自覚、感受性、感覚過敏性、完璧性
    - HSPだと上がる
    - まだありません<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 思想
- 努力ではカバーできないか、少なくともしづらいような「根本的な性能」にフォーカスしたいです
- EQは使いたくないです
    - 能力の解像度が粗いので、もう少し細かくしたいです
    - 自分の内面の理解と、外界の理解は分けたいです
- 創造性を測る指標もほしいです
