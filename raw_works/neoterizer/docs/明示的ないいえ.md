# 明示的ないいえ
# いいえの旨を明示的に示すこと。
# Explicit No.

- これにより:
    - Before: 回答対象は宙ぶらりんのままで進捗がわからない
    - After: 回答対象に「いいえ」の旨がついたので中止とわかる
- つまり回答対象をタスクとみなし、クローズに導くための情報を明示的に出させるということです。これによりクローズしていない宙ぶらりんなタスクの数を減らせ、認知資源の節約ができます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
