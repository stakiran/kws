# オシアブル
# 推すことができる、の意。
# Oshiable.

# 概要
- 推しとは:
    - 特定の人間のファンになること
- ファンになるとは:
    - 行動として以下三つが当てはまる
        - 1. コンテンツとして消費する
        - 2. 理想を押し付ける
        - 3. おかず（性的または性的を問わず妄想のネタとして使うもの）にする
    - 2つ以上当てはまるとき、その人のファンである。この状態を[* ステルスなファン]という
        - 一つの場合は当てはまりません。たとえば1.はゴシップでも満たせます<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
    - 行動している様を本人に直接的・間接的に伝えたとき、[* オープンなファン]という
- オシアブルとは、本人がオープンなファンになってもいいと認めている状態を指す
    - 通常はアイドルなど、それを生業とする役割の場合にオシアブルとなる
    - 日常では、人気者が特定の人（達）に対して、自身をそのように消費してもいいと認めることがあり、これもオシアブルと言える

# ビジネスとオシアブル
- オシアブルになることでエンゲージメントを高めることができる
    - 例: 社内アイドル
        - たとえば[インスピレンサー(Inspirencer)](インスピレンサー(Inspirencer).md)のように社員全員に高頻度でズバズバ言う者がいるとして、かつオシアブルにした場合、社員らはこのインスピレンサーを「推す」ことができる。推せるなら堂々と会話のネタにできるし、会話に出せばより広がっていく
        - アイドルのように「推してもいい」ことを認めるのである
        - 組織も結局は人であり、組織に対するエンゲージメントや[従業員エンゲージメント]も人に紐づくものである。であれば、推すことを認めることは、それらを高めるのに適した手段だろう
