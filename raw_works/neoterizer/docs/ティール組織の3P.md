# ティール組織の3P
# [ティール組織]の本質を整理したもので、Party、Protocol、Peaceの略。

# 3P
- Party
    - 小集団
    - それ単体で業務を完結できる、多くとも20人以内の集団
    - たとえば人事権を持っており、自分たちだけで新しい人材を雇える
- Protocol
    - プロトコル
    - 紛争解決方法含む「組織のあり方」が文書化されており、しかも動的な更新と（インターネットへの）オープンな公開もしている
- Peace
    - 資本主義にとらわれない経営やビジョン
    - そのようなあり方で顧客や市場に選ばれることを目指す
    - たとえば競合にも無償で啓蒙をする。社長自ら無償で競合他社で講演するなど
