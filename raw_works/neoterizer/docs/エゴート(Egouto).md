# エゴート(Egouto)
# チームプレイやチームのやり方あり方には馴染めないが、自律的に動けて報連相もできる者。
# [エゴセントリック]性が強いが、自律的に動ける者。
# Egoistic Auto.

# 例
- 自分の生活リズムにこだわりを持つASD
- 組織のリズムに合わせられないADHD、その他[典型的な多様性](典型的な多様性.md)
# いずれにせよ「組織のリズムに合わせられない」点で共通している。なぜ合わせられないかが違う<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 例2:
- ストイック型
    - 「うるせえ、私は朝6時に起きて夜23時に寝る生活をしなければならないんだよ」
- 怠慢型
    - 「毎日同じ時間に出社とかだるいよ、やる気出たときに働けばいいじゃん」
# 自分のやり方こそが正義だと本気で思っている。
# また、それで仕事や副業や趣味などで安定的に結果を出せている（結果を出せる程度に行動しきれる＝自律性がある）。

# 概要
- エゴート成立の要件
    - エゴートであること。自分なりの過ごし方があり、それをストイックに踏襲し、しかし自律的で結果を安定的に出せること
    - エゴートの[アダプター(Adapter)](アダプター(Adapter).md)がいること。エゴートと外界（チームや顧客）をつなぐ橋渡し役が存在すること
- エゴートの扱い方3原則
    - 1 [デリファメーション(Delifirmation)](デリファメーション(Delifirmation).md)
    - 2 [🙏ギャランティ](🙏ギャランティ.md)よりもベストエフォート
    - 3 同期よりも非同期
