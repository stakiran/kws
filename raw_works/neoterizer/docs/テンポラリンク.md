# テンポラリンク
# 一時的なリンク集のこと。あるいは単にそのために一時的なリンクを行うこと。
# Temporary Links.

# 応用
- Cosenseにおいて、生成AIに読ませるために一時的につくる
    - たとえば今、私は「既読」という概念をどう扱っているかを調べるために、一時的なリンク集をつくりました。検索結果から一件ずつコピペしました（スクリプトで自動化もできるでしょう）。するとCosenseの[Export For AI]が使えるようになります。言い方を変えると、エクスポートを行うために、一時的なリンク集をつくったのです<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
        - [https://gyazo.com/64fdbfbf1a4c9e7ac2b86cd3ff5dd26b][https://gyazo.com/6993a13ddf8a7507e95502efb3b73441]
    - 生成AIに読ませたあとは、ページ名も変更して、そのまま検討をつづけられます
