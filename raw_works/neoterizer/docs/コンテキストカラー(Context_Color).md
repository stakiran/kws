# [タスク]の種類ごとに色を事前に決めておき、その色を設定して俯瞰することで視認性を上げること。
# 特にその時に使う色のこと。

# 開発秘話
- このような概念は言わずと知れていましたが、名前が不在でした。それもそのはずで、スマートな名前をつけるのが明らかに難しかったからです。私は知的生産者としてこの難問に挑み、コンテキスト（文脈）の言葉を取って、この名前にしました。一般的には種類、カテゴリー、プロジェクト、タグやラベルといった言葉が使われがちですが、そうではなく、結局タスクを文脈ごとに分けているわけなので、コンテキストで良かったのです
