# はおすや
# はじめた、おわった、すすめた、やめたの略で、[ABN(Activity Based Networking)](ABN(Activity_Based_Networking).md)での投稿時に使える「端的な観点」。

<img src="https://gyazo.com/7daa598ea346245e1b4cc2500523e5a9.png" />


- はじめた
    - 新たに始めた場合
- おわった
    - 終わった場合
- すすめた
    - 新たに始めたわけではなく、終わってもやめてもなく、しかしすすめた場合
    - 「すすんだ」ではなく「すすめた」です。主観的に「進めた」という自覚があれば、もう投稿してもいいのです<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- やめた
    - 終わったのではなくやめた場合
