# テキシメージ(Teximage)
# 文章のスクショなど「文章を写した画像」に、その文章の内容をステガノグラフィー等で埋め込んだもの。
# Text Attached Image.

# 背景
- 文章を「たしかな実体を持つ情報」として静的に保存するにはPDFが適しているが、以下のデメリットがある
    - パースしづらい
    - 形式が重たい
- よりかんたんに保存するなら画像が適しているが、以下のデメリットがある
    - パースできない
        - OCRを使えば可能ですが処理時間と品質に限界があります<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
- そこで第三のあり方を考えます

# 概要
- テキシメージとは、文章の「画像」と「文書そのもの」を両方保持するもの
- つくりかた:
    - 1: 文章をテキストでつくる
    - 2: 1のスクショを撮るなどして画像化する
    - 3: 2に1を埋め込む
        - たとえば[ステガノグラフィー]で埋め込む
- 配布時は画像の形で配布し、必要なら専用ツールで（画像から）テキストを取り出して利用する

# 応用
- スライドに適用する:
    - 1: スライドを画像化
    - 2: スライドを「テキストで解説したもの」を作成する
    - 3: 1に2を埋め込む
