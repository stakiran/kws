# QWINCS
# 主な[🙏コラボレーション](🙏コラボレーション.md)ツールを指す[📝アクロニム](📝アクロニム.md)で、Q&A、Wiki、Issues、Note、Chat、Sticky boardの略。
# 読み方は「クインクス」。

# 開発秘話
- Sticky boardはMiroなどデジタルホワイトボードを指す
    - Stickyだと「付箋」のニュアンスしかないのだが、無論ボードは付箋以外も多様なコンテンツを置けるため不適切だ。しかし、アクロニムの語感のため、仕方なく採用した
- IssuesはGitHub Issuesのようなツールを指す
    - 本当はチケットやトピックなどもっと抽象化するべきだが、このジャンルの開祖は明らかにGitHub Issuesであるので、敬意も込めてこれで良いとした
- ChatはSlackやTeamsやDiscordなどビジネスチャットである
    - これらはコミュニケーションツールであって[🙏コラボレーション](🙏コラボレーション.md)ツールではないが、実態として2025年現在でもまだまだコラボレーションツールとしても使われているため無視できず、加えた
