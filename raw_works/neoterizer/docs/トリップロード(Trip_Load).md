# トリップロード(Trip Load)
# 旅やイベントのような移動的・喧騒的・探索的な高負荷がかかる状況のこと、またそのような負荷。

# 背景
- 出張や大型イベントには適性がある
    - 1 移動の負荷、会場の喧騒という負荷、初来訪や初体験の地や場所を探索する負荷など、慢性的に高負荷がかかるからだ
    - 2 また予定、交通、宿泊先などの手配、調整、管理といった作業も要求され、これら作業をこなす負担も無視できない
    - タフな人は耐えられるし、通常も（連続は無理だがなんとか）耐えられるが、脆い人はそうじゃない
        - [フラジャイル(Fragile)](フラジャイル(Fragile).md)
- このような負担は旅でかかる負荷と似ている
    - よってTrip + Load（負荷）という形で名前をつけて捉えることにした<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 概要
- トリップロードは、精神的な刺激ではなく物理的な刺激の量と幅が圧倒的ということである
    - 精神的な刺激は扱っていない
    - ちなみに精神的な刺激に状況は関係ありません。極論FXで全財産が溶けたとき、パートナーが死んだとき、人身事故の現場を間近で目撃してしまったときなどは一瞬で高負荷になります。もう少し難しく議論すると、物理的な刺激は主に「環境」から起こりますが、精神的な刺激は「イベント」によって起こります<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>

# 意義
- [多様性の実装](多様性の実装.md)に使える
    - トリップロードを強要しないあり方を実装できる
    - 実装するためには議論できねばならず、議論するためには言及が必要で、言及には名前が必要である。トリップロードという名前で捉えたので、実装までできるというわけだ<a href="sta.md"><img src="https://gyazo.com/a0a22d2fc5cf4fb2525db091fb66594b.png" alt="sta" width="16"/></a>
