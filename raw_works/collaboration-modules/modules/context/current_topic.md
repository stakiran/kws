# Current Topic
**Current Topic** とは、会議や会話において今現在扱われている話題を指す。

その会議や会話の目的、アジェンダ、各人の思い、場の空気といったパラメーターが重要に思えるが、そうではない。最も重要なのは今現在、実際に扱われている話題なのである。この点を強調するためにあえて Current Topic と名付けている。

Current Topic の概念により、私達は「今現在の話題」そのものを扱えるようになる。たとえば、目的やアジェンダに沿わない話題を脇に置く「パーキングロット」は知られており、会議中はパーキングロットなるエリアを設けてそこに書き留めるだろう。同様に、Current Topic のエリアをつくって、今現在の話題を書くことができる。全員がここを見ながら話すことで、今現在の話題を意識できるようになる。
