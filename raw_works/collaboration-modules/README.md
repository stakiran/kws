# Collaboration Modules

コラボレーション（複数人の協業）で繰り返し起きる問題に対する、概念的な道具集。AI エージェントに読み込ませて対話する使い方を想定して設計されている。

著者: [sta](https://github.com/stakiran)

## 使い方の例

チームで意思決定が遅い、会議が多すぎる、非同期コミュニケーションがうまくいかない——そんなとき、このリポジトリを AI エージェントに読み込ませて壁打ちすると、問題に対応するモジュールが提案され、論点が整理される。

```
あなた: うちのチーム、会議が多すぎて作業時間が取れません。何かいいやり方はありますか？
AI: いくつかのモジュールが参考になりそうです。
    - 「静的な会議」: 開催や進行の仕方を固定して会議コストを下げる
    - 「RAIS」: 定期的に非同期で情報共有し、会議の代替にする
    - 「コミュニケーションタイム」: 関係構築の時間を別途確保し、会議の目的を分離する
```

## モジュール一覧

全 97 モジュール。**Conceptware**（使える道具としての手法・考え方）と **Contextware**（暗黙的な文化・慣習の言語化）の 2 タイプがある。
