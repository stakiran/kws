## 読者向け
前提として自分向けの作業場を公開しているだけであり、読者向けのわかりやすさは重視していない。しかし、私自身にとってわかりやすいものは、おそらく読者にとってもわかりやすいはずである。少なくとも潔癖に整備されているさまは見て取れよう。

読みたい方は wiki/index.scb から辿ると良い。なお .scb ファイルは独自文法であり、VSCode に [vscode-scb - Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=stakiran.vscode-scb) をインストールすれば快適に読み書きできる。

おそらく生成 AI と一緒に読み進めていくのが良いだろう。CLAUDE.md を見てもわかるとおり、作者本人も Claude Code と一緒に作業している。
